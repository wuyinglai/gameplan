# 余烬商队（Ember Caravan）

> 一款「商队 RPG + 战术卡牌战斗 + 订单护送 + 长线资源管理 + 天气风险事件」的游戏项目。
>
> 核心体验：护住人、护住车、护住货、控制补给、管理士气、判断天气、决定是等待、绕路、撤退还是硬闯。

## 与《杀戮尖塔》的核心差异

| 维度 | 杀戮尖塔 | 余烬商队 |
|------|---------|---------|
| 角色结构 | 单角色 | 初期三人小队，后期可扩展到 4 人 / 5 人出行，共用行动力 |
| 压力来源 | 血量 | 粮食、士气、车耐久、伤病、天数 |
| 地图 | 爬塔 | 大地图格子移动，按天计算 |
| 战斗目标 | 杀敌 | 保护货车、货物、订单 |
| 成长 | 经验值 | 旧文明记忆、金币强化、火种强化与固定记忆融合 |

## 世界观一句话

> 灰潮之后，旧世界的能源网络熄灭。玩家带领一支商队，携带一枚「余烬核心」，穿越灰色荒原，重新点亮沿途驿站，最终抵达失去电力的启明城——把一条熄灭的路重新点亮。
