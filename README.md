# 🔥 余烬商队（Ember Caravan）

> 一款「大地图商队 RPG + 战术卡牌战斗 + 订单护送 + 长线资源管理 + 天气风险事件」的游戏项目。
>
> 核心体验：护住人、护住车、护住货、控制补给、管理士气、判断天气、决定是等待、绕路、撤退还是硬闯。

---

## 📋 项目概览

| 项目属性 | 内容 |
|---------|------|
| **游戏名称** | 余烬商队（Ember Caravan） |
| **游戏类型** | 商队RPG / 战术卡牌 / 生存抉择 / 肉鸽 |
| **参考品类** | 《杀戮尖塔》+ 《FTL》+ 商队经营 |
| **核心玩法** | 三人小队卡牌构筑 + 大地图远征 + 资源管理 |
| **目标平台** | PC（首版Demo） / 后续扩展移动端、主机 |
| **开发阶段** | 原型验证 → 垂直切片 |

## 🎯 核心差异

与《杀戮尖塔》的核心区别：

| 维度 | 杀戮尖塔 | 余烬商队 |
|------|---------|---------|
| 角色结构 | 单角色 | **三人小队**，共用行动力 |
| 压力来源 | 血量 | **粮食、士气、车耐久、伤病、天数** |
| 地图 | 爬塔 | **大地图格子移动** |
| 战斗目标 | 杀敌 | **保护货车、货物、订单** |
| 成长 | 经验值 | **城市重建 + 古代记忆 + 金币强化** |

## 🗺️ 世界观一句话

> 灰潮之后，旧世界的能源网络熄灭。玩家带领一支商队，携带一枚「余烬核心」，穿越灰色荒原，重新点亮沿途驿站，最终抵达失去电力的启明城——把一条熄灭的路重新点亮。

这不是英雄拯救全世界的故事。这是几个人、一辆商队、一枚核心，在灰潮之后，把一条熄灭的路重新点亮的故事。

## 🎮 核心循环

```
接订单 → 采购补给/零件 → 规划路线 → 大地图移动
            ↓
    战斗 / 事件 / 营地 / 天气风险
            ↓
    抵达驿站/城市 → 治疗/修车/交货/休整 → 接下一单
```
