# 《余烬商队》系统设计总框架 v0.1

## 1. 核心定位

《余烬商队》不再按轻量肉鸽卡牌游戏设计，而是改为：

> 大地图商队 RPG + 战术卡牌战斗 + 订单护送 + 长线资源管理 + 天气风险事件。

玩家不是单纯打怪升级，而是带着人、车、货、补给和订单，在漫长危险的商路上前进。

核心体验是：

* 护住人
* 护住车
* 护住货
* 控制补给
* 管理士气
* 判断天气
* 决定是否等待、绕路、撤退、硬闯
* 通过技能、状态、装备和队友配合打出高爆发

游戏的味道应该是：

> 每一步都是一天，每一天都在消耗，每一次等待都在赌订单，每一次坏天气都可能是灾难，也可能是机会。

---

## 2. 时间与路线系统

### 2.1 时间单位

统一设定：

* 移动 1 步 = 消耗 1 天
* 休养 1 天 = 重伤恢复进度 +1
* 等待 1 天 = 天气推进 +1
* 订单期限按天计算
* 天气持续按天计算
* 补给消耗按天计算

这样所有系统都可以统一到“天数”上。

### 2.2 路线长度

路线按驿站分段。

每个驿站之间大约：

* 18–24 天 / 步

推荐路线结构：

| 路线类型     |    驿站数量 |        总距离 | 定位      |
| -------- | ------: | ---------: | ------- |
| 短线 / 教学线 |   1 个驿站 |    35–50 天 | 新手路线    |
| 标准线      |   2 个驿站 |    55–75 天 | 主力路线    |
| 中线       | 3–4 个驿站 |   80–120 天 | 有明显规划压力 |
| 长线       | 5 个以上驿站 | 130–180+ 天 | 真正远征    |
| 超长商路     | 7 个以上驿站 |     200+ 天 | 后期大订单   |

### 2.3 路线决策

玩家出发前需要判断：

* 到下个驿站大概多少天
* 要带多少补给
* 要带多少零件
* 是否少带货、多带生存物资
* 天气是否适合出发
* 订单期限是否允许等待
* 遇到坏天气是否硬闯、绕路或等待

---

## 3. 城市与驿站系统

城市和驿站是恢复、经营、修理、采购和决策节点。

### 3.1 城市功能

城市是完整恢复点，可以提供：

| 功能     | 说明                       |
| ------ | ------------------------ |
| 休养角色   | 花钱恢复生命，缩短重伤休养            |
| 治疗失能角色 | 第 3 次重伤后的角色可治疗救回         |
| 招募新人   | 替换失能角色，或补充后备角色           |
| 修理货车   | 花钱 + 零件恢复货车耐久            |
| 购买补给   | 自由采购                     |
| 购买零件   | 自由采购                     |
| 接订单    | 普通订单、危险订单、护送订单、加急订单      |
| 卖货     | 处理贸易收益                   |
| 打听情报   | 获取下一段路线风险、天气、Boss、特殊事件信息 |

### 3.2 驿站功能

驿站是半恢复点，功能少一些，价格更贵。

| 功能      | 说明                    |
| ------- | --------------------- |
| 基础休养    | 可恢复部分生命，但难以完全治疗严重失能   |
| 简单修车    | 可修一部分货车耐久             |
| 买补给     | 数量有限，价格较贵             |
| 买零件     | 数量较少                  |
| 临时招募    | 可招募较弱或临时队员            |
| 存货 / 卸货 | 暂时放下一部分货物             |
| 打听路线    | 获得下一段路的风险提示           |
| 天气预报    | 花钱查看未来天气趋势            |
| 等待天气    | 等待坏天气过去，但消耗补给、金钱和订单期限 |

城市是“大修整”，驿站是“续命点”。

---

## 4. 补给、零件、金钱与采购

### 4.1 补给

补给是生存资源。

基本规则：

* 每移动 1 天消耗 1 补给
* 补给可以在城市和驿站购买
* 补给不足时仍可移动，但会掉士气和生命

建议购买量：

| 路线目标         | 理论最低补给 |    推荐携带 |
| ------------ | -----: | ------: |
| 到下个驿站，约 20 天 |     20 |   25–30 |
| 标准线，约 60 天   |     60 |   75–90 |
| 中线，约 100 天   |    100 | 120–140 |
| 长线，约 150 天   |    150 | 180–220 |

缺补给惩罚：

* 补给为 0 时，每移动 1 天：

  * 士气 -1
  * 全队每人失去 2 点生命

如果士气很低，惩罚可加重：

* 士气 ≤ 2 时缺补给移动：

  * 士气 -1
  * 全队每人失去 3 点生命
  * 有概率获得疲惫状态

### 4.2 零件

零件是修车、修装备、技能消耗和特殊事件处理资源。

零件不随移动自动消耗，但用于：

* 修理货车
* 修理装备
* 修补师技能
* 特殊战斗修车
* 事件处理
* 车辆模块维护

推荐携带量：

| 路线目标  |  推荐零件 |
| ----- | ----: |
| 到下个驿站 |   3–5 |
| 标准线   |  8–12 |
| 中线    | 12–18 |
| 长线    | 20–30 |

零件应比补给更贵、更稀缺。

---

## 5. 士气系统

士气代表商队精神状态，是长期失败压力。

### 5.1 基础数值

* 初始士气：6
* 最大士气：10
* 士气归零：进入崩溃状态

士气不建议做成 100 点大数字。小数字更有压迫感。

### 5.2 士气变化

| 事件        |    士气变化 |
| --------- | ------: |
| 缺补给移动     |      -1 |
| 全队倒下      |      -2 |
| 角色失能      |      -1 |
| 重要货物被毁    |      -2 |
| 订单失败      | -1 到 -3 |
| 战斗撤退      |      -1 |
| 成功抵达驿站    |      +1 |
| 成功完成订单    | +1 到 +3 |
| 打赢强敌      |      +1 |
| 救助 NPC 成功 |      +1 |
| 城市休整      |      +1 |

### 5.3 士气归零

士气归零不一定立刻游戏结束，而是进入“崩溃状态”。

崩溃状态下：

* 不能接高风险订单
* 招募价格上升
* 部分队员可能拒绝出战
* 如果再次全队倒下，商队解散
* 如果无法恢复最低行动能力，则读档 / 游戏失败

真正失败不是一次失败，而是商队彻底组织不起来。

## 6. 角色生命、重伤与失能

### 6.1 初始生命

| 角色    | 初始生命 | 定位           |
| ----- | ---: | ------------ |
| 护路人   |   50 | 最能扛，保护队伍     |
| 荒野射手  |   38 | 输出高但较脆       |
| 修补师   |   42 | 中等生命，负责救场和修理 |
| 全队总生命 |  130 | 前期基础承压值      |

### 6.2 重伤机制

生命归零后，不立刻死亡，而是：

* 获得 1 次重伤
* 退出当前战斗
* 进入休养状态
* 需要等待若干天后恢复
* 恢复后生命为最大生命的 30%

推荐规则：

| 重伤次数    | 惩罚                       |
| ------- | ------------------------ |
| 第 1 次重伤 | 最大生命 -10%，休养 3 天         |
| 第 2 次重伤 | 最大生命 -20%，休养 4 天         |
| 第 3 次重伤 | 进入失能状态，不能继续战斗，必须到城市治疗或替换 |

### 6.3 不做永久死亡

角色不要永久死亡。

第 3 次重伤后进入“失能 / 离队治疗”状态：

* 到城市后可花钱治疗
* 可用医疗资源救回
* 可暂时招募新人替补
* 可长期保留失能角色，等以后有钱再治疗

这样惩罚很重，但不会让玩家因为一次失败直接弃档。

---

## 7. 全队倒下与失败处理

### 7.1 全队倒下

全队倒下后不直接游戏结束，而是触发重惩罚：

* 损失全部当前货物
* 损失全部补给
* 损失全部零件
* 商队退回最近驿站
* 所有角色强制进入休养状态
* 士气 -2
* 货车耐久降到危险值，例如 20%
* 当前订单大概率失败或严重降级

这会非常痛，但还能继续。

### 7.2 真正失败条件

可设为：

* 士气归零后再次全队倒下
* 全队倒下且无法恢复最低行动能力
* 商队没有补给、没有零件、没有可行动角色、没有金钱，且无法接任何补救任务
* 关键主线订单彻底失败

失败本质是：

> 商队解散，而不是某一次战斗输了。

---

## 8. 货车耐久系统

货车是商队的命根子，等同于第四名队友。

### 8.1 基础数值

* 货车最大耐久：100
* 耐久为 0 时：抛锚，无法继续移动，必须修到 20 以上

### 8.2 耐久损耗

不要每走一步都固定掉耐久，而是根据路况、天气、战斗和事件变化。

| 情况           |     货车耐久损耗 |
| ------------ | ---------: |
| 普通道路         |   每 5 天 -1 |
| 荒路 / 山路      |   每 3 天 -1 |
| 灰烬区 / 风暴区    | 每 1–2 天 -1 |
| 普通战斗受波及      |    -3 到 -8 |
| 特殊敌人攻击货车     |   -8 到 -15 |
| Boss 撞击 / 抢货 |  -15 到 -30 |
| 使用强力车载技能     | 消耗 5–20 耐久 |

### 8.3 耐久状态

| 耐久     | 状态 | 效果                     |
| ------ | -- | ---------------------- |
| 71–100 | 完好 | 正常                     |
| 41–70  | 受损 | 坏事件概率上升，部分车载技能效果下降     |
| 21–40  | 危险 | 移动额外消耗补给；战斗开始可能少抽 1 张牌 |
| 1–20   | 濒毁 | 无法接高价值护送订单；被攻击容易损失货物   |
| 0      | 抛锚 | 不能继续移动，必须修理            |

### 8.4 修理规则

建议基础规则：

* 野外急修：1 零件 = 修复 8 点耐久
* 修补师野外修理：1 零件 = 修复 10 点耐久
* 城市修理：1 零件 + 3 金 = 修复 15 点耐久
* 驿站修理：1 零件 + 5 金 = 修复 10 点耐久

修补师可通过技能或被动提高修车效率。

### 8.5 车载技能

部分强力技能可以消耗货车耐久。

例子：

* 车体冲撞：消耗 8 点耐久，对所有敌人造成伤害
* 拆板护卫：消耗 5 点耐久，为角色提供大量护甲
* 过载炮击：消耗 15 点耐久和 1 火药，对单体造成高伤害

核心抉择：

> 打赢眼前这场，但车可能撑不到下个驿站。

## 9. 战斗长度与基础数值

由于游戏转向 RPG 化，战斗可以比轻肉鸽更长。

### 9.1 行动力

* 基础每回合全队共用 3 点行动力
* 中后期极稀有方式可提升到 4
* 临时效果可达到 5
* 不建议轻易永久 5

行动力从 3 到 4 是巨大成长，必须有高代价或稀有条件。

### 9.2 战斗长度

| 战斗类型              |    推荐回合数 |
| ----------------- | -------: |
| 前期普通战斗            |   4–6 回合 |
| 中期普通战斗            |   6–8 回合 |
| 后期普通战斗            |  7–10 回合 |
| 精英战斗              |  9–12 回合 |
| 普通 Boss           | 12–16 回合 |
| 章节 Boss / 终局 Boss | 18–25 回合 |

### 9.3 敌方总血量

| 战斗类型    |    敌方总血量 |
| ------- | -------: |
| 前期普通战斗  |   90–130 |
| 中期普通战斗  |  160–230 |
| 后期普通战斗  |  260–420 |
| 前期精英    |  180–240 |
| 中期精英    |  280–380 |
| 后期精英    |  420–600 |
| 前期 Boss |  300–420 |
| 中期 Boss |  500–750 |
| 后期 Boss | 800–1200 |

后期 Boss 可以高血量，但必须允许玩家通过构筑打出高爆发，否则会变成刮痧。

### 9.4 玩家输出预期

| 阶段 | 普通回合输出 |  配合回合输出 |  爆发回合输出 |
| -- | -----: | ------: | ------: |
| 前期 |  18–26 |   30–45 |   50–70 |
| 中期 |  35–55 |   60–90 | 100–140 |
| 后期 |  60–90 | 110–180 | 220–350 |

高爆发必须依赖铺垫：

* 敌方有多种减益
* 我方有多个增益
* 队友辅助
* 装备触发
* 消耗特殊物品
* 使用高阶战技

## 10. 技能、记忆与状态系统

### 10.1 卡牌定位改为“记忆 / 战技”

卡牌不再是普通肉鸽奖励，而是角色掌握的记忆、战技、经验或特殊能力。

因此：

* 普通战斗不再稳定给卡牌
* 精英战斗给普通记忆或记忆碎片
* Boss / 重要剧情给稀有记忆、特殊战技或改写材料

### 10.2 技能类型

技能分为三类：

#### 基础技能

随时能用，数值一般。

例：

* 1 费，造成伤害并获得少量护甲

#### 条件技能

需要敌方减益或我方增益达到条件，效果明显更强。

例：

* 目标至少有 2 种负面状态，才能释放
* 我方角色拥有 2 种增益，技能强化

#### 终结技能

条件更苛刻，但可以打出高爆发。

例：

* 目标有 3 种以上减益
* 射手拥有“专注”
* 消耗 1 火药
* 造成大量伤害，击杀返还行动力

### 10.3 状态系统

#### 敌方负面状态

| 状态   | 效果               |
| ---- | ---------------- |
| 标记   | 受到射手技能额外伤害       |
| 流血   | 每回合受到伤害          |
| 破甲   | 护甲降低或受到攻击加伤      |
| 易伤   | 下一次受到伤害增加        |
| 燃烧   | 每回合伤害，可被油桶或火药强化  |
| 迟缓   | 下回合行动或伤害降低       |
| 灰烬污染 | 灰烬系天气 / 敌人相关特殊状态 |

#### 我方正面状态

| 状态   | 效果                |
| ---- | ----------------- |
| 守势   | 下次获得护甲提高          |
| 专注   | 下次攻击提高            |
| 备战   | 下回合额外抽牌或加伤        |
| 加固   | 本场战斗护甲效果提高        |
| 医疗准备 | 下次治疗提高            |
| 装填   | 射手下次攻击增强          |
| 过载   | 修补师或车载技能强化，但可能有代价 |

### 10.4 状态数量条件

可以设置状态数量门槛：

* 目标有 1 种负面状态：小加成
* 目标有 2 种负面状态：解锁强技能
* 目标有 3 种负面状态：解锁终结技
* 目标有 4 种以上负面状态：触发爆发倍率

我方也可以类似：

* 角色有 1 种增益：技能强化
* 角色有 2 种增益：附加额外效果
* 角色有 3 种增益：转为爆发技

### 10.5 道具消耗技能

部分技能需要消耗特殊物品。

例：

| 技能   | 消耗          | 效果        |
| ---- | ----------- | --------- |
| 燃烧瓶  | 1 桶油        | 给敌方全体添加燃烧 |
| 爆裂箭  | 1 火药        | 造成范围伤害    |
| 急救针  | 1 医疗包       | 立即治疗或缩短休养 |
| 临时护板 | 1 零件        | 提供大量护甲    |
| 过载炮击 | 1 火药 + 货车耐久 | 高额单体伤害    |

这样技能不仅消耗行动力，还消耗商队资源。

## 11. 卡牌升级与记忆改写

### 11.1 普通升级

普通升级是简单增强：

* 伤害 +2 到 +4
* 护甲 +2 到 +4
* 治疗 +2 到 +3
* 减少代价
* 增加少量附加效果

### 11.2 记忆改写 / 战技变体

中后期可以加入更有 RPG 味的改造系统。

一张卡可以改写成不同版本。

例：

原版：

* 重劈：2 费，造成 20 点伤害

轻装版：

* 重劈·轻装：1 费，造成 15 点伤害

压制版：

* 重劈·压制：2 费，造成 18 点伤害，并添加破甲

赌命版：

* 重劈·赌命：1 费，失去 4 生命，造成 22 点伤害

这类系统比单纯加数值更有构筑感。

---

## 12. 普通战斗、精英战斗与奖励

### 12.1 普通战斗奖励

普通战斗不稳定给卡牌。

奖励以维持商队为主：

| 奖励        |     概率 |
| --------- | -----: |
| 金钱        | 必给或高概率 |
| 补给        |   中高概率 |
| 零件        |    中概率 |
| 消耗品       |    小概率 |
| 记忆碎片      |   极低概率 |
| 完整记忆 / 战技 |   一般不给 |

普通战斗是“维持生存”，不是刷卡。

### 12.2 精英战斗奖励

精英战斗可给：

* 普通记忆 2 选 1
* 记忆碎片
* 稀有消耗品
* 较多金钱、补给、零件
* 小概率特殊装备或车辆模块

### 12.3 Boss 奖励

Boss 奖励不固定为“必给遗物”。

Boss 的核心奖励应该是：

* 保住关键货物
* 完成高价值订单
* 解锁城市功能
* 提高路线安全
* 获得声望
* 获得特殊记忆 / 改写材料
* 解除追击压力
* 获得剧情推进

Boss 的意义不是爆装备，而是影响路线、订单和商队命运。

---

## 13. 特殊战斗类型

战斗不只有“杀光敌人”一种胜利条件。

### 13.1 普通战斗

目标：

* 击败全部敌人

用途：

* 常规遭遇
* 野兽
* 普通强盗
* 灰烬感染者

### 13.2 伏击战

触发：

* 危险节点
* 低侦察
* 高价值货物
* 沙尘、浓雾、灰烬风暴等天气

规则：

* 敌人先手
* 第一回合玩家行动力 -1，或敌人获得先攻
* 货车可能初始受损

胜利：

* 击退敌人

额外压力：

* 如果拖太久，敌人开始抢货

### 13.3 撤退战 / 断后战

触发：

* 玩家主动撤退
* 遇到强敌
* 订单风险过高
* 天气恶劣

规则：

* 不要求杀光敌人
* 坚持指定回合数即可撤离

推荐回合：

| 阶段        | 坚持回合 |
| --------- | ---: |
| 前期        |    4 |
| 中期        |  5–6 |
| 后期        |  7–8 |
| Boss 追击撤退 | 8–10 |

撤退成功也会有代价：

* 损失少量补给
* 士气 -1
* 货物受损
* 货车耐久下降

### 13.4 护送战

目标：

* 保护货车
* 保护乘客
* 保护药品
* 保护净水核心
* 保护驿站仓库

失败：

* 护送目标耐久归零

奖励：

* 目标剩余耐久越高，奖励越高

### 13.5 抢回货物战

敌人目标不是杀人，而是抢东西。

规则：

* 敌人派搬运者靠近货车
* 搬运者接触货车后开始搬货
* 搬货需要 2 回合
* 成功后开始逃跑

胜利：

* 击退敌人
* 阻止关键货物被带走
* 或抢回关键货物

失败：

* 关键货物被带走

### 13.6 修车逃生战

触发：

* 货车耐久过低
* 特殊事件
* Boss 撞击
* 恶劣天气
* 路线事故

规则：

* 角色可消耗行动力和零件修车
* 修到指定耐久后可以撤离

例：

* 消耗 1 行动力 + 1 零件，恢复 12 点耐久
* 修补师可恢复 18 点耐久

### 13.7 坚持到援兵

适合城市、驿站和声望系统。

规则：

* 援兵倒计时 6–10 回合
* 玩家需要撑住
* 城市声望越高，援兵越快

胜利：

* 坚持到援兵到达

### 13.8 限时击杀战

规则：

* 敌人正在点燃货物、破坏车轮、召唤援军
* 玩家必须在指定回合内击杀关键目标

失败后果：

* 货物损坏
* 货车耐久大降
* 敌方援军加入

### 13.9 资源消耗战

规则：

* 每 2–3 回合损失补给、零件、货车耐久或士气
* 玩家需要尽快结束战斗

适合：

* 灰烬污染
* 虫群
* 风暴中的敌人
* 腐化商队残骸

---

## 14. Boss 设计方向

Boss 不以普通守路型为主。

Boss 核心问题应该是：

> 它为什么找你？它想抢什么？它会毁掉什么？你能不能谈判？

### 14.1 护送型 Boss

Boss 要抢某个关键货物。

例：

* 黑旗劫货队
* 药品掠夺者
* 贵金属劫匪

机制：

* Boss 派小怪搬货
* 小怪接近货车后开始搬运
* 玩家可以击杀搬运者夺回货物
* 关键货物被抢走则失败或订单严重降级

### 14.2 摧毁型 Boss

Boss 目标是摧毁关键物资或货车。

例：

* 灰烬巨兽攻击净水核心
* 机械怪物攻击旧文明电池
* 火焰敌人点燃燃油车厢

机制：

* Boss 有蓄力冲撞
* 玩家可用护路人挡下
* 修补师可临时加固货车
* 射手可打断蓄力

### 14.3 谈判型 Boss

Boss 不一定必须打。

例：

* 沙岭税帮
* 黑旗首领
* 山路军阀
* 路霸商团

玩家选择：

* 交钱
* 交货
* 用声望压价
* 欺骗
* 假装投降后伏击
* 拒绝并进入 Boss 战

### 14.4 阶段型 Boss

重要 Boss 可多阶段。

例：

第一阶段：抢货
第二阶段：破车
第三阶段：灭口

不同阶段改变目标和敌方行为。

### 14.5 追击型 Boss

Boss 在地图上形成压力。

规则：

* 每移动若干天，追击进度 +1
* 货物越贵，追击越快
* 玩家可以丢货、绕路、设伏、雇诱饵、主动迎战

追击 Boss 适合长路线，不一定每次都直接开战。

### 14.6 Boss 与订单绑定

订单决定 Boss 风险。

| 订单类型      | 可能 Boss       |
| --------- | ------------- |
| 护送药品      | 强盗抢药          |
| 运送贵金属     | 黑旗劫货队         |
| 运送旧文明电池   | 机械追击者         |
| 护送贵族 / 技师 | 绑架型 Boss      |
| 运送燃油      | 火焰 / 爆炸型 Boss |
| 运送净水核心    | 摧毁型 Boss      |
| 运送军械      | 谈判型军阀 / 劫匪    |

高价值订单应伴随更高 Boss 风险。

---

## 15. 天气系统

### 15.1 核心规则

不做昼夜系统，只做天气。

时间规则：

* 1 步 = 1 天
* 每段天气持续约 5–10 天
* 大多数时间是晴朗或轻微天气
* 严重天气低概率出现
* 灾害级天气极低概率，并有隐藏冷却
* 玩家可在城市 / 驿站等待天气过去
* 天气影响路线、事件、战斗、补给、货车耐久和奖励

### 15.2 天气概率

建议基础概率：

| 天气      |  概率 |
| ------- | --: |
| 晴朗      | 50% |
| 阴云 / 微风 | 18% |
| 沙尘      | 10% |
| 暴雨      |  8% |
| 高温      |  6% |
| 浓雾      |  4% |
| 寒潮      |  3% |
| 灰烬风暴    |  1% |

晴朗和轻微天气占大多数。寒潮、灰烬风暴这类严重天气概率很低。

### 15.3 天气持续时间

| 天气      |   持续时间 |
| ------- | -----: |
| 晴朗      | 6–10 天 |
| 阴云 / 微风 |  5–8 天 |
| 沙尘      |  4–7 天 |
| 暴雨      |  3–6 天 |
| 高温      |  5–9 天 |
| 寒潮      |  3–5 天 |
| 浓雾      |  2–4 天 |
| 灰烬风暴    |  2–4 天 |

### 15.4 隐藏冷却

严重天气有隐藏冷却，防止连续折磨玩家。

建议：

* 灰烬风暴出现后，未来 30 天内不会再次自然生成
* 寒潮出现后，未来 20 天内不会再次自然生成
* 连续两次坏天气后，下一次天气提高晴朗 / 阴云概率

玩家不看到具体冷却，只看到叙事提示：

> 风暴刚过，天象暂时趋稳。

### 15.5 天气预报

城市和驿站可提供天气预报。

| 服务       | 效果             |
| -------- | -------------- |
| 免费闲聊     | 知道当前天气还会持续大概多久 |
| 少量花费     | 查看未来 5 天天气趋势   |
| 较高花费     | 查看未来 10 天天气趋势  |
| 高声望 / 情报 | 提前知道是否有极端天气    |

预报不必 100% 精准，可以用趋势描述：

* 未来 5 天大概率晴朗
* 未来 10 天可能有沙尘
* 远处有灰烬风暴迹象

### 15.6 等待天气

城市 / 驿站可以等待。

等待 1 天：

* 消耗补给
* 消耗住宿费
* 重伤角色休养 +1 天
* 订单期限 -1 天
* 天气剩余天数 -1 天

玩家需要判断：

* 等天气安全，但消耗时间和补给
* 硬闯危险，但能赶订单期限
* 坏天气可能带来稀有资源

---

## 16. 天气强度分级

天气可分为普通版和强化版，后期可扩展灾害级。

### 16.1 沙尘

普通沙尘：

* 侦察 -15%
* 伏击概率 +10%
* 远程技能第一回合伤害 -10%

强沙尘暴：

* 侦察 -30%
* 伏击概率 +20%
* 天气期间无法准确识别未知节点
* 货车每 3 天耐久 -1
* 射手标记类技能有小概率失败或效果下降

沙尘结束后可能奖励：

* 被埋货物
* 旧路标
* 隐藏驿站
* 风沙掀开的遗迹入口

### 16.2 暴雨

普通暴雨：

* 泥泞路段增加
* 每移动 3–5 天，货车耐久额外 -1
* 燃烧效果 -50%
* 水源事件增加

洪水暴雨：

* 部分路线临时阻断
* 绕路天数 +5 到 +15
* 货车耐久损耗增加
* 有概率触发陷车 / 修车逃生战

暴雨奖励：

* 收集纯净水
* 水类订单收益提高
* 洪水退去后冲出旧货箱或遗迹入口

纯净水用途：

* 卖高价
* 完成城市订单
* 恢复士气
* 用于治疗 / 休养
* 在高温天气抵消部分补给压力

### 16.3 高温

普通高温：

* 每 4–5 天额外消耗补给
* 缺补给时士气额外下降
* 长战斗中容易疲惫

灼热荒日：

* 补给消耗明显提高
* 角色休养恢复量下降
* 战斗超过 6 回合后，全队获得疲惫

高温奖励：

* 太阳能旧设备更容易启动
* 干燥药材品质提高
* 某些灰烬兽活动减少

### 16.4 寒潮

普通寒潮：

* 休养效率下降
* 治疗效果 -10%
* 每 5 天额外消耗补给
* 部分野兽敌人变弱

极寒：

* 休养暂停或大幅下降
* 治疗效果 -20%
* 等待也消耗更多补给
* 补给不足时士气下降更快

寒潮奖励：

* 保存完好的药品
* 冰封仓库
* 腐化怪活动减少
* 稀有冷藏货物

### 16.5 浓雾

普通浓雾：

* 未知节点无法提前识别
* 伏击概率上升
* 奇遇概率上升
* 第一回合无法攻击部分后排目标

迷雾：

* 路线可能偏移
* 隐藏事件概率大幅提高
* 容易遇到神秘商人、失踪商队、旧文明信标
* 同时伏击概率提高

浓雾特点：

> 危险和奇遇一起上升。

### 16.6 灰烬风暴

灰烬风暴是本游戏标志性灾害天气。

灰烬风暴期间：

地图效果：

* 侦察大幅下降
* 伏击概率上升
* 每移动 1–2 天，货车耐久下降
* 普通事件部分替换为灰烬事件

战斗效果：

* 灰烬兽 / 灰烬感染者 / 灰烬化敌人获得强化
* 普通敌人可能获得灰烬污染状态
* 战斗拖得越久，环境伤害越明显

灰烬兽强化建议：

* 生命 +20%
* 攻击 +15%
* 每 3 回合获得 1 层灰烬狂化
* 死亡时有概率爆发灰烬污染

灰烬风暴奖励：

* 打赢灰烬相关战斗后，有概率获得灰烬材料
* 风暴结束后 3–5 天内，灰烬残骸事件概率提高
* 有概率发现旧文明遗迹入口
* 有概率发现迷失商队残骸
* 有概率获得特殊记忆碎片

灰烬材料用途：

| 材料     | 用途            |
| ------ | ------------- |
| 灰烬结晶   | 制作高级装备 / 车辆模块 |
| 灰烬残片   | 记忆改写材料        |
| 风暴核心   | 强化特殊技能        |
| 污染样本   | 卖钱或研究         |
| 特殊记忆碎片 | 合成稀有战技        |

灰烬风暴原则：

> 极危险，但值得冒险。

---

## 17. 特殊事件系统

特殊事件按类型分池。

### 17.1 资源事件

例：

* 废弃仓库
* 水源
* 残破货车
* 被埋货箱

可能奖励：

* 补给 +10 到 +30
* 零件 +2 到 +6
* 货车耐久 +10
* 特殊物品

风险：

* 伏击
* 污染补给
* 损坏货车

### 17.2 路线事件

例：

* 断桥
* 塌方
* 泥路
* 废弃隧道
* 旧路标

结果：

* 绕路 +5 到 +20 天
* 消耗零件修路
* 发现近路
* 降低下段伏击概率

### 17.3 商队事件

例：

* 队员争吵
* 伤员恶化
* 货物松动
* 车轮异响

结果：

* 士气变化
* 角色休养变化
* 货车耐久变化
* 触发小任务

### 17.4 NPC 事件

例：

* 旅人
* 难民
* 商人
* 猎人
* 逃兵

选择：

* 救助：消耗补给，获得士气 / 声望
* 交易：获得物资
* 拒绝：安全但可能失去声望
* 抢夺：获得资源，但降低声望或引来追击

### 17.5 订单事件

与当前货物绑定。

例：

* 药品泄漏
* 贵金属引来眼线
* 旧文明电池发热
* 护送乘客生病
* 燃油车厢不稳定

结果：

* 订单奖励提高或降低
* 触发 Boss 追击
* 需要消耗零件、药品或行动处理

### 17.6 遗迹事件

探索旧文明设施。

奖励：

* 特殊记忆碎片
* 装备
* 车辆模块
* 稀有材料
* 城市科技线索

风险：

* 机械敌人
* 污染
* 货车耐久损耗
* 角色受伤

### 17.7 天气事件

由天气触发。

| 天气   | 事件倾向          |
| ---- | ------------- |
| 晴朗   | 商人、旅人、路线情报    |
| 沙尘   | 被埋货箱、迷路、伏击    |
| 暴雨   | 洪水、泥坑、净水收集    |
| 高温   | 中暑、干裂水井、太阳能设备 |
| 寒潮   | 冻伤、冰封仓库       |
| 灰烬风暴 | 污染、风暴核心、迷失商队  |
| 浓雾   | 神秘人、错路、隐藏驿站   |

### 17.8 Boss 前置事件

用于让 Boss 不突然出现。

例：

* 发现黑旗标记
* 看到被洗劫的商队
* 货物被人做了记号
* 远处出现跟踪者
* 驿站老板警告你

处理结果可以影响 Boss：

* Boss 初始生命降低
* Boss 少一波小怪
* 可开启谈判选项
* Boss 更强
* 伏击开局
* 抢货速度更快

---

## 18. 订单系统

订单是路线、Boss、收益和风险的核心连接点。

### 18.1 订单类型

| 订单类型   | 特点                   |
| ------ | -------------------- |
| 普通订单   | 收益稳定，风险较低            |
| 加急订单   | 时间紧，奖励高，不适合等待天气      |
| 危险订单   | 时间宽松，但容易触发 Boss / 抢货 |
| 护送订单   | 保护乘客或特殊货物            |
| 高价值订单  | 收益高，引来强敌             |
| 城市复兴订单 | 推进城市发展和主线            |

### 18.2 订单期限

订单按天数限制。

例：

普通订单：

* 距离 60 天
* 期限 90 天
* 容错 30 天

加急订单：

* 距离 60 天
* 期限 70 天
* 奖励高，但几乎不能等天气

危险订单：

* 距离 60 天
* 期限 100 天
* 时间宽松，但容易触发 Boss

### 18.3 订单与天气

天气会影响订单决策：

* 坏天气等待会消耗期限
* 硬闯坏天气会损车、损人、损货
* 特定天气可提升某类订单收益

例：

* 暴雨可收集纯净水，提高水类订单价值
* 灰烬风暴后可获得灰烬材料，适合稀有材料订单
* 高温会提高补给压力，压缩加急订单容错

## 20. 当前定稿原则

目前框架定稿如下：

1. 游戏定位改为长线商队 RPG + 战术卡牌。
2. 1 步 = 1 天。
3. 地图路线拉长，每个驿站间隔约 18–24 天。
4. 城市负责完整恢复、治疗、招募、采购、修车、接订单。
5. 驿站负责有限恢复、补给、简单修理、天气预报和等待。
6. 补给按天消耗，可自由购买。
7. 零件用于修车、技能、事件和车辆模块。
8. 士气是长期失败压力，初始 6，最大 10。
9. 角色不永久死亡，第 3 次重伤后失能，可到城市治疗或替换。
10. 全队倒下后退回最近驿站，损失全部货物、补给和零件。
11. 货车耐久加入系统，最大 100，耐久为 0 时抛锚。
12. 普通战斗不再稳定给卡牌。
13. 卡牌改为“记忆 / 战技”，获得更稀缺。
14. 技能可要求敌方减益或我方增益达到条件才能释放。
15. 部分技能可消耗油、火药、医疗包、零件、货车耐久等物品。
16. 战斗可以有特殊胜利条件，不只是杀光敌人。
17. Boss 不以普通守路型为主，而是围绕货物、订单、谈判、抢夺、护送和追击设计。
18. 天气系统按天持续，大多数时候晴朗，恶劣天气低概率。
19. 灰烬风暴是标志性灾害，会强化灰烬系敌人，但也带来稀有材料。
20. 坏天气不是纯惩罚，而是高风险 + 特殊收益。
