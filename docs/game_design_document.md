## 1. 游戏概述

### 1.1 一句话描述
《余烬商队》是一款「小队卡牌构筑 + 商队远征 + 生存抉择」的肉鸽游戏。玩家带领三人小队和一辆货车，在灰潮之后的荒原大地图上前进，管理粮食、士气、车耐久和角色伤病，在限定天数内完成订单并抵达终点。

### 1.2 核心体验
- 护住人、护住车、护住货
- 控制补给与零件消耗
- 管理士气，判断是否休整
- 观察天气，决定等待、绕路、撤退还是硬闯
- 通过技能、状态、装备和队友配合打出高爆发

### 1.3 与《杀戮尖塔》的核心差异

| 维度 | 杀戮尖塔 | 余烬商队 |
|------|---------|---------|
| 角色 | 单角色 | **三人小队**，共用 3 点行动力 |
| 压力 | 血量 | **粮食、士气、车耐久、伤病、天数** |
| 地图 | 爬塔 | **大地图格子移动**，按天计算 |
| 战斗目标 | 杀敌 | **保护货车、货物、订单** |
| 成长 | 经验值 | **城市重建 + 古代记忆 + 金币强化** |

## 2. 世界观与叙事

### 2.1 故事背景
故事发生在**灰潮之后**。旧世界曾依靠庞大的能源网络维持运转，城市、驿站、灯塔都连接在这张网络上。后来灰潮来了——它像灰色的海一样漫过荒原，经过的地方，能源设施失效，灯塔熄灭，机械失控，城市失联。

灰潮最可怕的地方不是立刻杀死人，而是让一切系统慢慢失效。

> 灰潮来临时，先熄灭的不是灯，而是路。

### 2.2 核心目标
玩家带领的商队携带一枚仍能启动旧世界节点炉的**"余烬核心"**，穿越荒原，重新点亮沿途驿站，恢复旧能源网络的一部分，最终抵达失去电力供应的**启明城**，将核心接入主反应炉。

这不是英雄拯救全世界的故事。这是几个人、一辆商队、一枚核心，把一条熄灭的路重新点亮的故事。

### 2.3 关键概念

| 概念 | 说明 |
|------|------|
| **灰潮** | 灾变本身。像雾又像风暴，会让能源失效、野兽狂暴、方向混乱 |
| **余烬核心** | 旧世界遗留的高阶启动核心，可重新启动能源网络中的关键节点 |
| **节点炉** | 每座驿站的核心装置，核心接入后驿站永久激活 |
| **驿站回传** | 驿站网络恢复后的快速通行机制（非玄幻瞬移） |

## 3. 核心玩法

### 3.1 玩法循环 (Core Loop)

```
接订单 → 采购补给/零件 → 规划路线 → 大地图移动
            ↓
    遭遇：战斗 / 事件 / 营地 / 天气风险
            ↓
    抵达驿站/城市 → 治疗/修车/交货/休整 → 接下一单
```

### 3.2 时间与路线

- **1 步 = 1 天**：所有系统统一按天计算
- **路线按驿站分段**，每段约 18–24 天
- 路线类型：短线(35–50天) / 标准线(55–75天) / 中线(80–120天) / 长线(130–180+天)

### 3.3 主要系统

#### 3.3.1 战斗系统（详见 [N2.txt](../N2.txt)）

- **三人小队共用 3 点行动力**
- 每角色拥有独立牌组
- 防御体系：**屏障 → 护盾 → 护甲 → 生命**（四层结算）
- 状态系统：标记、流血、灼烧、灰蚀、迟滞、回刃、过载、蓄力、狂化
- 货车是"第四名队友"，有独立耐久（初始 85/100）

三名角色定位：

| 角色 | 定位 | 核心机制 |
|------|------|---------|
| **护路人** | 护盾/防守/护车/承伤 | 迎击、掩护、稳住阵线、盾反、重击、车队防护 |
| **荒野射手** | 标记/流血/迟滞/补射 | 标记、精确射击、放血箭、断筋箭、补射、灰核射击 |
| **修补师** | 治疗/修车/净化/过载 | 简易治疗、防护加固、车体修理、拆解破甲、灰烬净化、过载 |

#### 3.3.2 经济系统（详见 [N1.txt](../N1.txt)）

| 资源 | 城市价格 | 驿站价格 | 核心用途 |
|------|---------|---------|---------|
| **补给** | 2 金/个 | 3–4 金/个 | 每天消耗 1 个，缺补给则扣士气+扣血 |
| **零件** | 8 金/个 | 12–16 金/个 | 修车、事件处理、技能消耗 |
| **金钱** | — | — | 采购、治疗、修车、提升士气 |
| **火种** | 战略资源 | — | 城市重建、特殊强化、旧文明装置、剧情关键 |
| **余烬核心** | 主线货物 | — | 激活驿站、恢复城市能源 |

#### 3.3.3 士气系统（详见 [士气.txt](../士气.txt)）

- 士气范围：**0–10**，初始 6
- 士气归零触发**强制撤退**
- 同一场战斗/同一天士气最多扣 **-3**
- 恢复方式：酒馆小聚(+1)、商队宴席(+2)、豪华休整(+2)、打赢精英战(+1)

#### 3.3.4 天气与事件系统（详见 [第5类整合报告](../余烬商队_第5类整合总报告_v2.2_封档清洗版.md)）

- 天气按**"天气段"**生成，不是每天随机
- 天气段类型：普通天气(5–7天)、特殊天气(3–6天)、灰烬风(4–5天)、灰烬风暴(2–3天)
- 外部事件：受天气、地点、路线影响
- 内部事件：受商队状态（补给、士气、伤病、车耐久）驱动
- 事件冷却系统防止同一事件高频重复

## 4. 阶段设计

### 4.1 N3 新手阶段结构（详见 [N3.txt](../N3.txt)）

| 阶段 | 定位 | 核心目标 |
|------|------|---------|
| **N3.1** | 固定教学路段 | 教基础战斗、补给、零件、货车耐久 |
| **N3.2** | 半随机教学路段 | 教驿站功能、灰烬风暴避难、半随机路线 |
| **N3.3** | 城市周边经营教学 | 开放城市功能、工坊复产、古代记忆、短途订单 |
| **N3.4** | 开放世界展开 | 自由探索、小镇、遗迹、灰烬边缘、中程路线 |
| **N3.5** | 中程商路实践 | 正式进入开放商队玩法 |

### 4.2 第一版 Demo 目标（详见 [开发设计文档](../余烬商队_开发设计文档_v0.2_易读版%20(1).md)）

1. 从 5 个角色中选择 3 个
2. 进入大地图格子移动
3. 移动消耗天数和粮食
4. 进入战斗、事件、营地、补给点
5. 战斗中每角色有自己的手牌，三人共用 3 点行动力
6. 角色可能受伤、重伤、暂时不能上场
7. 补给点可治疗、招募、替换队员
8. 限定天数内抵达 Boss 格并击败
