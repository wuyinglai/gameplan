# 02 世界观与设定

本目录用于管理《余烬商队》的世界观、故事背景、阵营势力、城市地点、灰烬污染与旧文明、角色职业设定。

## 子目录

| 子目录 | 用途 |
|---|---|
| 世界基础信息/ | 灰潮、余烬核心、节点炉、驿站回传等核心世界设定 |
| 世界观补充/ | 补充设定、历史脉络、地理概况 |
| 阵营与势力/ | 商会、强盗集团、灰烬教团、旧文明残党等 |
| 城市与地点设定/ | 启明城、灰桥镇、驿站、遗迹等地点设定 |
| 灰烬污染与旧文明/ | 灰蚀、灰烬兽、旧文明技术、火种等 |
| 角色与职业设定/ | 初始三人、后续可招募角色、职业定位 |
