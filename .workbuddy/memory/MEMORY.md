## 游戏类型（已纠正）
- **不是开放世界**，是"大地图商队RPG + 战术卡牌战斗 + 订单护送 + 长线资源管理 + 天气风险事件"
- 大地图是格子（8×6或10×6），移动像下棋（上下左右走一格=1天+1补给）
- 战斗是回合制卡牌（共享行动力3点，每回合抽7牌，UI队列非棋盘）
- 3D开放世界是早期废弃方案（建不出模型），相关docx/xlsx全部丢弃不用

## 核心设定（来自第12类世界观报告 + 第6类，已按用户口头纠正更新）
- **余烬核心 = 激活密钥（不是动力源）**：旧文明遗留的"授权密钥"。插入每座驿站的【核心装置】后，才能启动该装置、永久激活驿站。本质是"钥匙"，不是"电池"。
- **驿站核心装置 = 「节点」（已定名）**：用户最终拍板就叫"节点"（不叫节点炉/枢炉）。每座驿站里被余烬核心(激活密钥)插入启动的母装置。激活节点=永久激活该驿站。
- **初始3角色（已确认为正式角色，非举例）**：护路人(68血/主承伤/护车/盾反)、荒野射手(40血/脆皮高输出/标记/流血/侦察)、修补师(46血/全能辅助/修车/加固/少量治疗)。不是筑垒者/寻烬者/斥候/鼓舞者。
- **记忆系统**：旧文明技术，让角色快速掌握强力能力，比普通学习好。完整获得无碎片。分类：攻击/防御/辅助/消耗。成长不靠经验/熟练度/碎片。
- **火种**：稀有战略资源。用于记忆强化、记忆融合、火种词条(技能加词条)。非碎片。
- **敌人3类**：强盗(含掠夺者车队)、野兽、灰烬兽。外加5个Boss。
- **主线**：携带余烬核心(激活密钥)，逐个激活5座驿站的枢炉(黑石/锈桥/白盐/北境中继/启明城)，最终重启启明城主反应炉。
- **当前权威世界观文件**：01_系统大类/第12类_.../第12类_剧情与世界观系统_v0.1.md
