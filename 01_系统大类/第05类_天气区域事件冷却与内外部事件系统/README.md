## 5.5 权威口径摘要

5.5 聚落驻留事件系统 v0.1 已生成。

聚落驻留事件用于处理城镇、城市、驿站内的普通随机事件，独立于野外事件池。当前基础版包括市场波动、谣言情报、医疗服务波动、维修服务波动、城市小治安、商队/旅人关系，不包含招募、灰兽潮预兆、城市建设、大危机和订单随机变价。
