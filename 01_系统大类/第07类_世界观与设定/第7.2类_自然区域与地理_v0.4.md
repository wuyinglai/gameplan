## 一、整体地图——3×3棋盘格

世界地图是一个**3×3的棋盘格子**，每个格子代表一个自然区域。

```
 ┌──────┬──────┬──────┐
 │ N2   │ N4   │ N8   │
 │ 枯林  │ 山地  │ 黑石  │
 │ 猎场  │ 峡道  │ 丘陵  │
 ├──────┼──────┼──────┤
 │ N5   │ N3   │ N7   │
 │ 风蚀  │ 河谷  │ 残田  │
 │ 沙砾  │ 水源  │ 牧野  │
 ├──────┼──────┼──────┤
 │ N1   │ N6   │ N9   │
 │ 灰原  │ 灰水  │ 灰烬  │
 │ 旷野  │ 湿地  │ 污染  │
 └──────┴──────┴──────┘
```

**布局逻辑**：
- **N3 河谷水源区**在正中心——它是连接最多区域的中枢
- **N1 灰原旷野区**在左下角——适合作为多数商队的起点/初始区域
- **N9 灰烬污染区**在右下角——最深处，最终区域
- **N5 风蚀沙砾区**在左上侧
- **N6 灰水湿地区**在右下侧

**区域间通行方式**：每个格子可以通过上下左右移动到相邻格子。也就是说：
- 从一个区域到另一个区域，必须经过它们之间的格子
- N1→N3 不能直达，中间要经过 N5 或 N6
- N3→N9 不能直达，中间要经过 N6 或 N7

**你改**：这个棋盘布局你觉得对不对？哪个格子要换位置？

## 二、各区域样貌（像素风，不写太细）

### N1 灰原旷野

**一句话**：灰色砾石平地，旧路标稀稀拉拉插着。

**像素风表现**：地砖用灰褐色方格，点缀少量碎石块。偶尔有一两根倾斜的铁杆（旧路标）。天空浅灰色。

### N2 枯林猎场

**一句话**：枯死的灰白树林，树干像骨头一样立着。

**像素风表现**：密集的枯树桩贴图，地面深褐色，落叶层。偶尔有野兽的小型轮廓在树影中移动。

### N3 河谷水源

**一句话**：灰绿色的河谷，一条河缓缓流过。

**像素风表现**：河流动画（浅蓝色，像素颗粒流），两侧是碎石滩和枯黄的灌木。河谷两侧有缓坡地形。

### N4 山地峡道

**一句话**：灰色峭壁间的窄路，头顶是一线天。

**像素风表现**：垂直的岩壁贴图（深灰色），路只有一格宽。部分路段有碎石堆挡住去路。风的效果（横向飘动的像素线）。

### N5 风蚀沙砾

**一句话**：黄褐色的沙砾地，风一直吹。

**像素风表现**：沙砾地面（黄褐色颗粒），被风蚀成奇怪形状的岩石。偶尔有风沙动画（半透明黄色遮罩）。

### N6 灰水湿地

**一句话**：灰白色的水面和泥沼，雾气弥漫。

**像素风表现**：浅灰色水面，深灰色泥滩，枯树桩露出水面。雾的动画效果，视线受限。偶尔冒泡的动画。

### N7 残田牧野

**一句话**：一道道旧田垄延伸到远处，偶尔有倒塌的农舍。

**像素风表现**：平行的田埂纹理（土黄色线），间或有废弃农用机械的锈蚀轮廓。天空比别处稍亮一点。

### N8 黑石丘陵

**一句话**：黑色的碎石丘陵，旧矿洞张着黑口。

**像素风表现**：深黑色/深灰色丘陵起伏，采石场的方形切面纹理。矿洞口是纯黑色的方块，可能冒着若有若无的灰烟。

### N9 灰烬污染区

**一句话**：灰白色的粉土地面，空中飘着灰色的絮状物。

**像素风表现**：灰白色地面（最浅的灰色），黑色岩石碎片。灰色飘浮颗粒（污染物动画）。偶尔能看到废墟建筑的残骸轮廓。

## 三、区域间路线关系

| 路线 | 说明 |
|---|---|
| N1 ↔ N5 | 灰原进入风蚀区的通道 |
| N1 ↔ N6 | 灰原进入湿地的通道 |
| N5 ↔ N2 | 风蚀区通向枯林 |
| N5 ↔ N3 | 风蚀区进入河谷中心 |
| N2 ↔ N4 | 枯林通向山地峡道 |
| N4 ↔ N3 | 山地峡道进入河谷中心 |
| N4 ↔ N8 | 山地通向黑石丘陵 |
| N3 ↔ N6 | 河谷中心通向湿地 |
| N3 ↔ N7 | 河谷中心通向牧野 |
| N7 ↔ N8 | 牧野通向黑石丘陵 |
| N7 ↔ N9 | 牧野通向灰烬污染区 |
| N6 ↔ N9 | 湿地通向灰烬污染区 |

**关键路径**：
- 从 N1（起始区）到 N9（最终区）的常规路线：N1→N6→N9（最短）或 N1→N5→N3→N6→N9（绕路）
- 从 N1 到 N3 中心区：N1→N5→N3 或 N1→N6→N3

## 四、区域间过渡

两个格子区域之间的过渡带是几格像素渐变的边界——不是突然切屏，而是走一小段边缘地带，地面纹理逐渐变化。

例：
- N1→N5：灰褐色砾石逐渐混入黄褐色沙粒 → 沙粒越来越多 → 进入N5
- N6→N9：灰色湿地水面逐渐变白变干 → 地面变成灰白色粉末 → 进入N9

这样的过渡带大约走**1~2步（天）**就能通过。
