# 9×9 地图设计（建议版）

以下是我自己设计的9×9棋盘地图，每个区域占2-6格，形状不规则。你过一下看哪不对。

```
    列1 列2 列3 列4 列5 列6 列7 列8 列9
行1 │ N5│ N5│   │   │   │   │ N7│ N7│   │
行2 │ N5│ N5│ N2│   │   │   │ N7│ N8│   │
行3 │   │ N2│ N2│ N4│ N4│   │   │ N8│ N9│
行4 │   │   │   │ N4│ N3│ N3│   │ N8│ N9│
行5 │   │ N1│   │   │ N3│   │   │   │ N9│
行6 │ N1│ N1│   │   │   │ N6│ N6│   │   │
行7 │ N1│   │   │ N6│ N6│   │   │   │   │
行8 │   │   │   │   │   │   │   │   │   │
行9 │   │   │   │   │   │   │   │   │   │
```

## 区域大小

| 区域 | 格数 | 位置说明 |
|---|---|---|
| N1 灰原旷野 | 5格 | 左下，行5-7，列1-2 |
| N2 枯林猎场 | 4格 | 左中上，行2-3，列2-3 |
| N3 河谷水源 | 4格 | 正中，行4-5，列4-5 |
| N4 山地峡道 | 4格 | 中上，行3-4，列4-5 |
| N5 风蚀沙砾 | 4格 | 左上角，行1-2，列1-2 |
| N6 灰水湿地 | 4格 | 中下，行6-7，列5-6 |
| N7 残田牧野 | 4格 | 右上，行1-2，列6-7 |
| N8 黑石丘陵 | 4格 | 右中，行2-4，列7-8 |
| N9 灰烬污染区 | 3格 | 最右，行3-5，列9 |

## 连通关系

相邻格子之间可以通行。大致路径：

| 路线 | 意思 |
|---|---|
| N1 ↔ N2 | 从灰原往上走进入枯林 |
| N1 ↔ N5 | 从灰原往左上方进入风蚀区 |
| N1 ↔ N6 | 从灰原往右进入湿地 (经过空格的通道) |
| N2 ↔ N4 | 从枯林向右进入山地 |
| N2 ↔ N5 | 枯林与风蚀区相邻 |
| N4 ↔ N3 | 从山地下行进入河谷 |
| N3 ↔ N6 | 从河谷向下进入湿地 |
| N3 ↔ N8 | 从河谷向右进入黑石丘陵 |
| N7 ↔ N8 | 牧野与丘陵相邻 |
| N8 ↔ N9 | 从丘陵向右进入灰烬区 |
| N7 ↔ N9 | 从牧野向右进入灰烬区 |
| N4 ↔ N8 | 山地向右连接丘陵 |

## 关键路径

- N1(起点) → N3(中心)：N1→N2→N4→N3 或 N1→N6→?→N3（需探索）
- N1(起点) → N9(最终)：经过N3→N8→N9 或 N3→N7→N9
