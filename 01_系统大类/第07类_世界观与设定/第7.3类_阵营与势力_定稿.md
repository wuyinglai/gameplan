# 《余烬商队》第 7.3 类：阵营与势力

> 文档状态：定稿  
> 所属大类：第 07 类 世界观与设定系统  
> 本篇内容：阵营体系总览——余烬商会、敌对势力（盗匪、灰渊信徒、灰兽）。

---

本篇记录游戏中已明确的阵营势力。中立阵营（旧文明遗民、灰烬教团、工匠会等）及各城商会分会详情留待后续补充。

## 一、余烬商会（可加入）

余烬商会是九城中唯一一个曾在各城设有分部的商业组织。九城时代它一度覆盖整片大陆——从灰原旷野到灰烬污染区的边缘，都有余烬商会的商队在路上。

灰潮之后，城市之间的商路中断，余烬商会的各城分会失去了联系。但每个分会都没有解散——它们在各自的城市周边继续运转，依靠城市附近的城镇、驿站和村庄维持着局部的贸易网络。

分会之间的商路断绝了。灰原城分会不知道黑石城的分会还在不在。枯林城分会不知道粮野城的分会还有没有人管事。

玩家的到来，将改变这一切。

### 加入商会

玩家不需要"加入"余烬商会——商队本身就是余烬商会的一部分。或者说，商队是流动的分会。

每到达一座有分会的城市，玩家可以与该城的商会负责人接触。分会之间失联多年，负责人会认出商队的商会身份，并提供帮助——前提是商队证明了自己有能力走通商路。

### 分会与声望

各城分会彼此独立，各有各的负责人和库存。玩家在某城分会的声望高了，该城分会才会提供更好的价格、更多的订单、更优惠的服务。声望通过完成该分会发布的任务、为该城带货等方式提升。

### 分会分布

| 城市 | 分会状态 | 说明 |
|---|---|---|
| 灰原城 | 活跃 | 分会正常运转，负责人健在 |
| 枯林城 | 活跃 | 规模较小，依赖猎人贸易 |
| 鱼水城 | 活跃 | 分会较富裕，水源贸易利润好 |
| 粮野城 | 活跃 | 分会与当地农场关系紧密 |
| 海盐城 | 活跃 | 分会掌握海盐专营 |
| 火油城 | 小型 | 分会人少，主要做烬油生意 |
| 药泽城 | 小型 | 分会闭塞，货物进出不易 |
| 黑石城 | 活跃 | 分会规模最大，财力最雄厚 |
| 灰烬城 | 无 | 城市半废弃，已无商会活动 |

## 二、盗匪（敌对·常见）

各路强盗、流寇、劫匪的统称。不是统一的组织，而是多个独立匪帮的集合。匪帮之间互相抢地盘、抢货物，并非铁板一块。

### 常见匪帮类型

| 类型 | 特点 | 主要出没 |
|---|---|---|
| 路匪 | 三五成群的小股强盗，劫掠落单商队 | 各处道路 |
| 匪帮 | 有组织、有据点的武装团伙，多达数十人 | 枯林猎场、风蚀沙漠边缘 |
| 流寇 | 没有固定据点的流窜匪徒 | 各处，随机出现 |

### 对玩家的态度

绝大多数时候是敌对的。但如果玩家的商队实力够强（人数多、护卫强），一些小型匪帮会选择避让。少数匪帮可以谈判——交"过路费"换平安通过。

---

## 三、灰渊信徒（敌对·极罕见）

崇拜灰烬母兽的狂热教派。他们认为灰潮不是什么灾难，而是"净化"——母兽将清除旧文明的一切，然后在灰烬上建立新的秩序。

灰渊信徒的人数极少，但每一个都极其危险。他们不怕死——不如说他们渴望为母兽献身。他们会在商道上设伏，有时用毒药暗算，有时直接正面袭击。

玩家在正常游戏中几乎不会遇到灰渊信徒。他们只在特定剧情、特定地点才会出现。​​灰渊信徒与盗匪最大的区别在于：盗匪为财，灰渊信徒为信仰。如果你遇到一队什么都不抢、只想杀人的袭击者——那你遇到的就是灰渊信徒。

---

## 四、灰兽（敌对·野兽）

灰烬兽和灰渊兽的统称。

| 类型 | 来源 | 危险性 |
|---|---|---|
| 灰烬兽 | 被灰潮感染的普通野兽 | 常见，越靠近灰烬污染区越多 |
| 灰渊兽 | 从灰潮中凝聚诞生的原生型怪物 | 罕见，比灰烬兽危险得多 |

灰兽不是有组织的势力，但它们对人类商队的威胁是持续性的。越往深处走，灰兽越多、越强。灰渊信徒中有人声称能与灰渊兽沟通——但从未被证实。
