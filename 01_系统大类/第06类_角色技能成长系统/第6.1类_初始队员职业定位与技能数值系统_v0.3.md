# 《余烬商队》第 6.1 类：初始队员、职业定位与技能数值系统 v0.3

> 状态：第 6 类第一阶段报告。  
> 本稿用于确定初始三名队员、职业定位、能力评级、基础技能数值、战斗状态接口和战斗节奏目标。  
> 本稿已根据最新口径修正角色血量、能力评级、技能费用、技能数值、护甲流、标记、盾反、穿甲、敌人意图标签、屏障系统、前后期敌人数值参考、Boss 战回合目标、后期技能增强方向和高条件爆发技能口径。  
> 本稿中与旧第 4 类表冲突的内容，以本稿为准。

## 一、6.1 的定位

第 6.1 类用于确定《余烬商队》初始三名队员的基础战斗结构。

本节主要解决：

1. 初始三名队员是谁；
2. 三名队员的生命值与能力评级；
3. 三名队员的职业定位；
4. 三名队员的基础技能牌、费用与前期数值；
5. 后期增强版技能的数值方向；
6. 护盾、护甲、屏障、穿甲、标记、流血、迟滞、盾反等核心战斗状态接口；
7. 普通战、精英战、Boss 战的节奏目标；
8. 敌人意图标签如何展示；
9. 后期高条件爆发技能的设计边界；
10. 旧版本中与当前口径冲突的内容如何废弃。

本节不处理后期招募、替补、记忆技能、火种成长、队员长期伤病与休养。这些内容放到后续 6.2、6.3 处理。

第 6.1 的目标是先把"初始三人 + 初始技能 + 基础战斗口径"定稳。  
后续所有第 6 类成长、伤病、远行能力，都要建立在这一节之上。

## 二、当前总原则

第 6.1 不再强行沿用第 4 类旧表中的角色血量与部分旧技能名。  
第 4 类旧表作为历史参考，但凡是与本节最新口径冲突，以本节为准。

当前统一原则如下：

| 项目 | 当前口径 |
|---|---|
| 初始队员 | 护路人、荒野射手、修补师 |
| 总生命 | 不要求刚好 150，合理范围 130–170 |
| 当前总生命 | 154 |
| 能力评级 | 只使用 A / B / C / D |
| 回刃 | 废弃旧名，统一改为盾反 |
| 破甲 | 废弃旧名，统一改为穿甲 |
| 穿甲 | 伤害类型，无视护甲 |
| 标记 | 可叠加，但叠加的是触发次数，不叠加伤害倍率 |
| 护盾 | 当回合临时防御，数值较高，回合结束默认清空 |
| 护甲 | 本场战斗内长期防御，可叠加，战斗结束清零 |
| 屏障 | 旧文明 / 火种级高级防御，不是普通角色技能 |
| 护甲流 | 可以作为一种正式流派存在，不再只是低存在感辅助 |
| 治疗 | 收敛，但不能低到没有意义 |
| 修补师 | 全能辅助，不是输出位，不是强奶妈，不是状态清除专家 |
| 普通战 | 约 5 回合左右 |
| 精英战 | 约 10 回合左右 |
| Boss 战 | 通常 15 回合以上，强 Boss 可 20 回合以上；速杀流可以更快，但不能按弱小怪节奏设计 |
| 后期普通技能 | 相对前期大约翻倍 |
| 后期核心输出 / 特攻 | 可达到前期 2–3 倍或更高 |
| 极高条件爆发 | 可以存在，但必须条件很难满足，适合后期记忆技能 / 火种技能 / 终局构筑 |

## 三、能力评级标准

第 6.1 统一使用 A / B / C / D 四档评级。

| 等级 | 含义 |
|---|---|
| A | 高，是角色核心优势 |
| B | 中等，有稳定作用 |
| C | 有一点，但不多 |
| D | 基本没有 / 完全没有 |

不使用 A+、A-、B+、C- 等加减评级。  
评级只用于快速定位角色能力，不用于精确数值计算。

## 四、初始三人生命值

当前建议生命值如下：

| 角色 | 生命 | 定位 |
|---|---:|---|
| 护路人 | 68 | 主承伤、防护、护车、盾反 |
| 修补师 | 46 | 全能辅助、修车、加固、少量治疗 |
| 荒野射手 | 40 | 脆皮高伤害、标记、流血、侦察 |
| 合计 | 154 | 位于 130–170 合理范围内 |

### 4.1 护路人为什么是 68

护路人是初始队伍中最能扛的角色，必须让他的血量优势明显。

护路人的职责包括：

- 承受敌方直接伤害；
- 给自己和队友提供护盾；
- 保护货车；
- 使用盾反惩罚攻击者；
- 在危险遭遇中稳定队伍。

如果护路人血量只比修补师高一点，他的主承伤定位会不明显。  
因此护路人生命定为 68。

### 4.2 修补师为什么是 46

修补师是全能辅助，不是前排，也不是半个坦克。

修补师需要比射手略稳，因为他承担商队维护功能；但不能太高，否则会模糊护路人的主防御地位。

修补师职责包括：

- 修车；
- 加固；
- 少量治疗；
- 提供临时护盾；
- 在关键时刻补救队伍和货车状态。

因此修补师生命定为 46。

### 4.3 荒野射手为什么是 40

荒野射手是脆皮高伤害角色。

他的职责包括：

- 高输出；
- 标记敌人；
- 施加流血；
- 施加迟滞；
- 补刀；
- 远行中发现痕迹和敌情。

射手生命最低，是为了让玩家明确感受到保护输出位的必要性。  
因此荒野射手生命定为 40。

## 五、初始三人能力评级表

| 角色 | 生存 | 输出 | 防护 | 治疗 | 修车 | 情报 | 状态处理 |
|---|---|---|---|---|---|---|---|
| 护路人 | A | B | A | D | D | D | D |
| 荒野射手 | C | A | D | D | D | A | C |
| 修补师 | B | D | B | B | A | D | D |

### 5.1 护路人评级说明

护路人是防护核心。

| 能力 | 评级 | 说明 |
|---|---|---|
| 生存 | A | 血量最高，负责承伤 |
| 输出 | B | 不是主输出，但有横身拦截、重击、盾反等补伤害手段 |
| 防护 | A | 核心能力，负责护盾、护车、稳阵 |
| 治疗 | D | 没有治疗能力 |
| 修车 | D | 不负责修车 |
| 情报 | D | 不负责侦察 |
| 状态处理 | D | 不负责清除异常状态 |

护路人的关键词是：撑住、保护、护车、盾反。  
他不是纯输出角色，但也不是完全没有伤害。

### 5.2 荒野射手评级说明

荒野射手是输出和情报核心。

| 能力 | 评级 | 说明 |
|---|---|---|
| 生存 | C | 脆，但不是完全不能挨打 |
| 输出 | A | 初始队伍最高输出角色 |
| 防护 | D | 不提供护盾或保护 |
| 治疗 | D | 没有治疗 |
| 修车 | D | 不负责修车 |
| 情报 | A | 远行中负责痕迹、伏击、敌情判断 |
| 状态处理 | C | 能施加标记、流血、迟滞等战斗状态，但不清除异常 |

荒野射手的关键词是：高伤害、标记、流血、补刀、侦察。  
他有很高的战斗价值，但需要护路人和修补师保护。

### 5.3 修补师评级说明

修补师是全能辅助。

| 能力 | 评级 | 说明 |
|---|---|---|
| 生存 | B | 比射手稳，但不是前排 |
| 输出 | D | 基本不承担输出职责 |
| 防护 | B | 可以提供护甲、护盾和货车保护，但不如护路人 |
| 治疗 | B | 有治疗，但治疗量收敛 |
| 修车 | A | 核心能力 |
| 情报 | D | 不承担侦察职责 |
| 状态处理 | D | 当前基础定位不做状态清除专家 |

修补师的关键词是：修车、加固、少量治疗、补救。  
他不是强奶妈，不是输出角色，也不是主防御角色。

## 六、基础战斗结构接口

第 6.1 使用第 4 类战斗系统中的基础结构作为底层接口。

| 项目 | 规则 |
|---|---|
| 战斗类型 | 回合制卡牌战斗 |
| 我方单位 | 护路人、荒野射手、修补师、货车 |
| 敌方单位 | 野兽、强盗、灰烬兽、精英、Boss |
| 行动力 | 每回合 3 点，全队共享 |
| 抽牌 | 每回合抽 7 张 |
| 手牌上限 | 无上限 |
| 回合结束 | 默认弃掉未使用手牌 |
| 角色移动 | 不做 |
| 敌人移动 | 不做 |
| 站位 | UI 队列，不是棋盘 |
| 胜利 | 敌人全灭 / 敌方逃跑导致战斗结束 / 特殊目标完成 |
| 失败 / 撤退 | 队员全倒、货车归零、士气归零、玩家主动撤退、逃跑成功等接第 3 类规则 |

共享行动力的意义是：玩家不能每回合什么都做。  
攻击、护车、治疗、修车、解状态、逃跑准备都要取舍。

## 七、敌方队列与敌人意图接口

第 4 类战斗表中已经定过：虽然取消前后排，但敌人仍然按列表排列，并可使用"一维相邻关系"。

### 7.1 敌方队列

敌方按列表排列，例如：

| 顺序 | 敌人 |
|---|---|
| A | 荒原强盗 |
| B | 荒原强盗 |
| C | 野兽 |
| D | 灰烬兽 |

敌人之间可以有"一维相邻"关系，用于：

- 燃烧扩散；
- 某些连锁技能；
- 某些群体状态；
- 敌人队列排序；
- 敌方逃跑 / 阻挡表现。

### 7.2 敌人意图标签的定位

敌人意图标签不是简单的"敌人类型标签"，也不是只能有一个。  
它用于告诉玩家敌人接下来大概想做什么，让玩家能做出决策。

敌人可以同时拥有多个标签，例如：

> 攻击角色 + 中毒攻击  
> 攻击货车 + 抢货准备  
> 防御 + 指挥  
> 蓄力 + 灰蚀攻击

标签分为两类：

| 类型 | 含义 |
|---|---|
| 行为意图 | 敌人准备做什么，如攻击、防御、逃跑、抢货、召唤 |
| 攻击效果 / 能力提示 | 敌人的攻击会附带什么，如中毒、灰蚀、流血、穿甲、缠绕 |

中毒、灰蚀、流血、穿甲等不应被写成独立的"攻击意图"。  
它们更准确地说是"攻击附带效果"或"敌人攻击能力"。

### 7.3 敌人意图标签表

| 标签 | 类型 | 说明 | 例子 |
|---|---|---|---|
| 攻击角色 | 行为意图 | 准备攻击我方队员 | 狼准备扑咬荒野射手 |
| 攻击货车 | 行为意图 | 准备攻击货车耐久 | 强盗准备砸车轴 |
| 抢货准备 | 行为意图 | 几回合后准备抢货，不一定是下回合 | 劫货手正在靠近货仓，2 回合后尝试抢货 |
| 正在逃跑 | 行为意图 | 敌方准备逃走 | 受伤强盗准备撤离 |
| 防御 | 行为意图 | 敌人准备给自己护盾 / 护甲 / 屏障 | 持盾强盗举盾 |
| 蓄力 | 行为意图 | 准备强攻击，可能下回合或数回合后释放 | 灰烬兽正在聚集灰火 |
| 召唤 | 行为意图 | 准备召唤小怪或援军 | 巢穴兽呼唤幼体 |
| 指挥 | 行为意图 | 给队友增益或影响逃跑意愿 | 匪首命令喽啰集火货车 |
| 中毒攻击 | 攻击效果 | 攻击命中后施加中毒 | 毒蛇咬人，命中后施加中毒 |
| 流血攻击 | 攻击效果 | 攻击命中后施加流血 | 狼撕咬，命中后施加流血 |
| 灰蚀攻击 | 攻击效果 | 攻击命中后施加灰蚀 | 灰烬兽爪击带灰蚀 |
| 穿甲攻击 | 攻击效果 | 攻击无视护甲 | 重弩强盗射击无视护甲 |
| 缠绕 | 攻击效果 / 控制 | 命中后施加迟滞或限制行动 | 藤蔓兽缠住护路人 |

### 7.4 示例：毒蛇意图

毒蛇的意图可以显示为：

| 标签 | 含义 |
|---|---|
| 攻击角色 | 准备咬一名队员 |
| 中毒攻击 | 这次攻击如果命中，会施加中毒 |

展示给玩家时可以理解为：

> 毒蛇准备咬人。若攻击突破护盾或造成至少 1 点伤害，则目标中毒。

### 7.5 示例：劫货手意图

劫货手的意图可以显示为：

| 标签 | 含义 |
|---|---|
| 抢货准备 | 正在接近货仓 |
| 攻击货车 | 可能先打货车或干扰货物 |

抢货不一定是下回合立刻发生。  
它可以是：

> 2 回合后抢货；  
> 蓄力完成后抢货；  
> 货车护盾被打穿后抢货；  
> 玩家不打断时抢货。

因此报告中不能写死"抢货 = 下回合抢货"。

## 八、战斗节奏目标

第 6.1 不直接重做完整敌人系统，但需要明确角色技能数值对应的战斗节奏。

| 战斗类型 | 目标回合数 | 说明 |
|---|---:|---|
| 普通小怪战 | 约 5 回合 | 主要是旅途消耗战，不宜拖太久 |
| 较难普通战 | 6–8 回合 | 敌人配置更麻烦，但仍不应太长 |
| 精英战 | 约 10 回合 | 要有压迫感和阶段感 |
| Boss 战 | 通常 15 回合以上 | 强 Boss 可 20 回合以上；速杀流可更快，但设计基准不能太短 |

Boss 战不应按普通战节奏设计。  
如果 Boss 只打 6–8 回合就结束，会缺乏压迫感和记忆点。

Boss 可以有：

- 阶段变化；
- 强攻击蓄力；
- 小怪召唤；
- 货车压力；
- 灰蚀 / 中毒 / 穿甲等特殊攻击；
- 剧情机制；
- 特殊胜利或撤退条件。

普通小怪战不是每次只有一个敌人。  
敌人数量不设硬上限，应按敌人强弱、类型和战斗主题决定。

| 敌人类型 | 数量设计 |
|---|---|
| 特别强的敌人 | 可以只出现 1 个 |
| 普通敌人 | 常见 2–3 个 |
| 弱小敌人 | 可以 4 个以上 |
| 狼群 / 鼠群 / 小型灰烬兽群 | 应该多个出现 |
| 精英战 | 可以 1 个精英 + 小怪，也可以单精英 |
| Boss 战 | 可以单 Boss，也可以 Boss + 召唤物 / 阶段小怪 |

例子：

| 遭遇 | 合理配置 |
|---|---|
| 狼群 | 4–6 只弱狼 |
| 强盗巡逻队 | 2–3 个强盗 |
| 强盗头目 | 1 个头目 + 2 个喽啰 |
| 灰烬兽幼体群 | 3–5 个低血量敌人 |
| 重甲强盗 | 1 个高护甲强敌 |
| Boss | 单体或阶段性召唤 |

### 9.3 前期敌人伤害参考

旧表中普通野兽前期普通攻击为 3–5，在当前角色血量和护盾数值下已经明显偏低。  
第 6.1 建议调整为：

| 敌人类型 | 普通攻击 | 强攻击 | 说明 |
|---|---:|---:|---|
| 弱小野兽 | 6–8 | 10–12 | 单体弱，但数量多 |

## 十、技能数值总基准

当前第 6.1 使用以下数值基准：

| 类型 | 前期基础值 | 后期增强方向 |
|---|---:|---:|
| 1 费普通攻击 | 9–12 | 18–24 |
| 1 费带状态攻击 | 7–9 + 状态 | 14–18 + 状态增强 |
| 1 费单体护盾 | 14–16 | 28–36 |
| 1 费群体护盾 | 每人 7–8 | 每人 14–18 |
| 1 费货车护盾 | 16–18 | 32–42 |
| 1 费治疗 | 8–10 | 16–22 |
| 1 费护甲 | 约 8 | 16–22 |
| 1 费穿甲 | 8–10 | 18–24 |
| 2 费普通攻击 | 18–24，按角色定位区分 | 40–55 |
| 2 费射手高伤 | 22–28，条件满足后更高 | 55–80，条件满足后 80–120 |
| 2 费特攻 | 满足条件后约 30+ | 普通后期 80–120；强构筑条件满足后 150–250 |
| 极高条件爆发 | 初始阶段不开放 | 后期 / 终局可达 300–500，必须满足极难条件 |

核心原则：

> 第 6.1 的数值不是低强度磨血，而是高影响回合制。每张牌都应对局势产生明显影响，普通战斗目标仍控制在 5 回合左右。

后期技能不是只做小幅成长。  
普通技能可按前期约 2 倍成长；核心爆发、条件特攻、过载组合等可以达到前期 2–3 倍甚至更高。玩家经过构筑和资源投入后，打出的强力技能必须让 Boss 血条产生明显变化，否则长战斗会变成疲劳磨血。

后期角色技能可以通过：

- 技能升级；
- 装备；
- 记忆技能；
- 火种强化；
- 车厢模块；
- 职业成长；
- 特定状态层数；
- 目标身上的多种负面状态；
- 过载；
- Boss 战机制窗口；

获得增强。

## 十一、后期高条件爆发技能口径

第 6.1 的初始技能不开放极高爆发，但需要为后期系统预留设计口径。

后期可以存在"条件极难满足，但伤害极高"的技能。这类技能通常不属于初始基础技能，而属于：

- 记忆技能；
- 火种技能；
- 高级职业成长；
- 高级装备技能；
- 车厢模块联动技能；
- Boss 战特殊机制技能；
- 终局构筑技能。

### 11.1 高条件爆发的条件类型

高条件爆发不能无脑释放，必须要求玩家提前构筑或承担代价。

可用条件包括：

| 条件类型 | 示例 |
|---|---|
| 多负面状态 | 目标同时拥有标记、流血、迟滞、中毒、灰蚀等多种状态 |
| 高层数状态 | 目标拥有高层数标记 / 流血 / 灰蚀压力 |
| 长回合准备 | 玩家花费 3–6 回合叠状态、过载、找牌 |
| 资源代价 | 消耗火种、稀有零件、特殊弹药、车厢充能 |
| 风险代价 | 使用者受到内伤、重伤风险、货车过载损伤 |
| Boss 窗口 | 只能在 Boss 破防、蓄力失败、阶段转换时释放 |
| 职业联动 | 需要三名队员的状态或技能组合同时满足 |

### 11.2 高条件爆发的伤害范围

这类技能可以非常高，但必须罕见、难达成、代价明显。

| 类型 | 后期伤害方向 |
|---|---:|
| 普通后期强技能 | 80–120 |
| 条件特攻 / 核心爆发 | 150–250 |
| 终局高条件爆发 | 300–500 |
| 剧情机制级爆发 | 可更高，但必须绑定 Boss 机制或特殊剧情 |

例子：

| 技能概念 | 条件 | 效果方向 |
|---|---|---|
| 灰烬处刑箭 | 目标同时有标记、流血、迟滞、灰蚀 | 造成 180–250 伤害 |
| 火种贯穿 | 消耗 1 个完整火种，目标处于破防窗口 | 造成 300+ 伤害 |
| 终局猎灰重箭 | 目标为灰烬兽 Boss，目标身上有高层数标记和灰蚀 | 造成 300–500 伤害 |
| 车厢主炮过载 | 消耗车厢充能和高级零件，货车受损 | 造成 250–400 伤害 |
| 全队协同爆发 | 三名队员各自满足一个前置状态 | 造成极高伤害或打断 Boss 阶段 |

### 11.3 高条件爆发的体验目标

高条件爆发的目的不是让玩家随便秒怪，而是让玩家在 Boss 战或终局长战中获得明确反馈：

> 我花了多个回合叠状态、承担代价、抓住窗口，终于打出这一击，Boss 血条必须明显下降。

如果玩家花了五六回合甚至十几个回合准备一个强力技能，结果 Boss 只掉十分之一甚至更少，会非常不爽。  
因此后期强力技能必须和 Boss 血量匹配。

设计原则：

| 原则 | 说明 |
|---|---|
| 普通技能稳定成长 | 后期基础技能约为前期 2 倍 |
| 核心技能明显强 | 后期核心输出可达前期 2–3 倍 |
| 高条件技能可极高 | 满条件时可达到 300–500 |
| 条件越难，伤害越高 | 不靠无脑数值，而靠构筑难度换爆发 |
| 不能破坏普通战 | 极高爆发不应在普通小怪战频繁使用 |
| Boss 需要防爆发机制 | Boss 可有阶段锁血、护盾、召唤、防御窗口等机制 |

## 十二、防御系统：护盾、护甲、屏障、穿甲

### 12.1 护盾

护盾是战斗中的短期防御资源。

| 项目 | 规则 |
|---|---|
| 持续 | 默认持续到本回合结束 |
| 作用 | 先于生命承受伤害 |
| 获取来源 | 护路人技能、修补师技能、道具、车辆牌 |
| 定位 | 立即挡伤害，短期防御 |

### 12.2 护甲

护甲是战斗内长期防御资源。

| 项目 | 规则 |
|---|---|
| 持续 | 本场战斗内跨回合保留 |
| 战斗结束 | 清零，不能带入下一场战斗 |
| 是否可叠加 | 可以叠加 |
| 是否设硬上限 | 不设硬上限 |
| 定位 | 可形成"叠甲流" |
| 克制方式 | 穿甲、状态伤害、攻击货车、多目标压力、Boss 机制 |

护甲不是低存在感的附属防御，而是一种可以形成流派的战斗资源。  
玩家可以在一场战斗中通过防护加固等手段持续叠甲，但这些护甲只存在于当前战斗，战斗结束后清零，不会带入下一次战斗。

同费用下，护甲数值可以约为护盾的三分之二。  
例如 1 费单体护盾约 12–16，则 1 费单体护甲可以约 8。

### 12.3 屏障

屏障是比护盾和护甲更高级的防御资源。

屏障来自旧文明装置、火种能量、高级道具或特殊剧情机制，不是普通初始角色技能的常规产物。

| 项目 | 规则 |
|---|---|
| 定位 | 高级防御 / 旧文明防御 |
| 常规来源 | 一次性屏障发生器、高级屏障发生器、残旧旧文明护罩、车载短效屏障 |
| 高级来源 | 火种屏障、旧文明装置 |
| 是否普通技能 | 否 |
| 作用 | 先于生命承受伤害，类似高级护盾 |
| 特点 | 数值高、稀有、通常需要道具或资源代价 |

旧表中已有屏障参考：

| 来源 | 屏障 | 代价 / 说明 |
|---|---:|---|
| 一次性屏障发生器 | 10 | 20–30 金币 |
| 高级一次性屏障发生器 | 18 | 45–60 金币 |
| 残旧旧文明护罩 | 20 | 稀有道具 |
| 车载短效屏障 | 20–30 | 消耗装置充能 |
| 火种全队护罩 | 全队 +50，货车 +50 | 消耗 1 个完整火种，需要旧文明装置 |

火种屏障定位：

> 绝境翻盘 / Boss 战救命 / 长线行商保命。

屏障不能作为普通角色基础技能频繁出现，否则会压过护盾和护甲系统。

### 12.4 穿甲

穿甲是伤害类型，不是持续状态。

| 项目 | 规则 |
|---|---|
| 是否无视护甲 | 是 |
| 是否削弱护甲 | 否 |
| 是否是状态 | 否，是伤害类型 |
| 是否无视护盾 | 默认不无视护盾 |
| 是否无视屏障 | 默认不无视屏障 |

默认规则：

> 穿甲无视护甲，但不默认无视护盾和屏障。

如果目标有护盾或屏障，穿甲伤害仍先被护盾 / 屏障吸收；如果目标只有护甲，则穿甲直接打生命。

## 十三、中毒 / 流血与防御接口

第 4 类战斗表中已定过中毒、流血与防御的关系。第 6.1 继承这一逻辑。

| 情况 | 是否施加中毒 / 流血 |
|---|---|
| 攻击被护盾完全挡住 | 不施加 |
| 攻击被屏障完全挡住 | 不施加 |
| 攻击打破护盾 | 施加 |
| 攻击打破屏障 | 施加 |
| 攻击穿透 1 点伤害 | 施加 |
| 攻击被护甲减伤 | 仍然施加 |
| 已有中毒后续伤害 | 不被护盾、护甲、屏障抵挡 |
| 已有流血后续伤害 | 不被护盾、护甲、屏障抵挡 |

最终口径：

> 护甲不能阻止中毒和流血。护盾和屏障只有完全挡住攻击时，才能阻止负面进入。只要破盾、破屏障，或穿透 1 点伤害，中毒 / 流血就会施加。

## 十四、状态系统接口

### 14.1 标记

标记由荒野射手提供。

新口径：

> 标记可以叠加，但叠加的是触发次数，不叠加伤害倍率。

| 标记层数 | 效果 |
|---:|---|
| 1 层 | 目标下一次受到我方直接伤害时，伤害 +25%，然后消耗 1 层 |
| 2 层 | 目标接下来两次受到我方直接伤害时，每次伤害 +25%，每次消耗 1 层 |
| 3 层 | 目标接下来三次受到我方直接伤害时，每次伤害 +25%，每次消耗 1 层 |

明确禁止旧理解：

- 2 层不是 +50%；
- 3 层不是 +75%；
- 标记不是无限叠倍率。

建议保留最低收益：

> 每层标记触发时，伤害 +25%，最低额外 +3。

### 14.2 流血

流血由荒野射手提供。

| 项目 | 规则 |
|---|---|
| 触发时机 | 目标行动时 |
| 每层效果 | 造成 4 点内伤 |
| 触发后 | 减少 1 层 |
| 定位 | 持续伤害 |

### 14.3 中毒

中毒通常由敌人攻击施加，也可由特殊环境或灰烬事件引发。

| 项目 | 规则 |
|---|---|
| 每层效果 | 每层 2 点内伤 |
| 额外效果 | 每层治疗效果 -10% |
| 治疗削弱上限 | 最多 -50% |
| 定位 | 长期削弱恢复能力 |

中毒不是单纯伤害状态，它还压制治疗。  
因此中毒敌人的威胁不只在当回合，而在后续持续损耗。

### 14.4 迟滞

迟滞由荒野射手提供，也可由敌人控制技能提供。

| 项目 | 规则 |
|---|---|
| 每层效果 | 行动失败率 +10% |
| 层数上限 | 5 |
| 消耗方式 | 行动判定后减少 1 层 |
| 定位 | 控制、拖慢敌人、提高逃跑率 |

迟滞也会影响逃跑率。第 4 类旧表中已有：逃跑加成可按敌方平均迟滞值提高。

### 14.5 灰蚀

灰蚀是灰烬系状态。

| 项目 | 规则 |
|---|---|
| 层数爆发 | 3 层时爆发 |
| 爆发伤害 | 8 点灰蚀伤害 |
| 爆发后 | 清空灰蚀 |
| 来源 | 灰烬兽、灰烬天气、特殊区域、Boss 技能 |

### 14.6 盾反

盾反由护路人提供。

| 项目 | 规则 |
|---|---|
| 旧名 | 回刃 |
| 正式名 | 盾反 |
| 触发条件 | 使用后本回合受到敌方直接伤害 |
| 效果 | 对攻击者造成反击伤害 |
| 触发次数 | 每次使用最多触发 1 次 |

盾反不再和"回刃"同时存在。  
回刃作为旧名废弃。

护路人是防护核心。

重击是护路人的补伤害技能，不是队伍主输出技能。  
即使后期成长，重击也不应压过荒野射手的核心输出和高条件爆发。

荒野射手定位：脆皮高伤害、标记、流血、迟滞、侦察。

### 16.1 精确射击处理

旧版中的"精确射击"不再作为基础 1 费技能保留。

处理方式：

| 旧技能 | 当前处理 |
|---|---|
| 精确射击 1 费基础攻击 | 删除 |
| 高费射击定位 | 由会合射击、猎灰重箭承担 |

这样射手技能组更有特色，不再只是普通攻击牌堆叠。

### 16.2 会合射击说明

会合射击是射手的标记联动高伤技能。

| 情况 | 前期伤害 |
|---|---:|
| 无标记 | 22 |
| 有标记 | 22 + 6 = 28，并消耗 1 层标记 |

它必须明显强于护路人的重击。  
因为荒野射手是输出核心，护路人只是补伤害角色。

### 16.3 猎灰重箭说明

猎灰重箭必须保留。

它是射手对灰烬兽的特攻技能。

| 目标 | 前期伤害 |
|---|---:|
| 普通敌人 | 20 |
| 灰烬兽 | 20 + 12 = 32 |

猎灰重箭让射手在灰烬兽战斗中有明显价值。

后期猎灰重箭可以成为灰烬兽 Boss 战中的核心技能。  
如果配合标记、过载、灰烬兽弱点、火种强化等机制，终局版本可以进入 300–500 的高条件爆发区间，但不能作为普通无条件输出频繁使用。

## 十七、修补师技能表

修补师定位：全能辅助、修车、加固、少量治疗。

修补师不是输出角色，也不是状态清除专家。  
他的技能要体现"补救"和"维护"，而不是打伤害。

| 技能 | 费用 | 类型 | 目标 | 前期基础效果 | 后期增强方向 |
|---|---:|---|---|---|---|
| 简易治疗 | 1 | 治疗 / 止血 | 我方单体 | 回复 9 生命，并清除 1 层流血；不能治疗重伤 | 回复 18–24，清除更多流血或附带少量护盾 |
| 防护加固 | 1 | 护甲 / 加固 | 我方单体 | 目标获得 8 护甲 | 16–24 护甲 |
| 车体修理 | 1 | 修车 | 货车 | 有普通零件：恢复货车 14；无零件：恢复货车 5 | 有零件 28–40；无零件 10–15；受零件品质、工具、车厢影响 |
| 拆解穿甲 | 1 | 穿甲 | 敌方单体 | 9 穿甲伤害 | 18–24 穿甲伤害 |
| 临时补强 | 1 | 护盾 / 补救 | 我方单体 / 货车 | 队员 12 护盾；货车 15 护盾 | 队员 24–32 护盾；货车 35–50 护盾 |
| 过载 | 2 | 强化 / 代价 | 指定一张牌 | 指定一张牌；该牌下次使用时数值效果 +100%；使用后，该牌拥有者受到 3 点内伤 | 可减少代价、提高增幅、附加抽牌或保护 |

### 17.1 简易治疗说明

简易治疗回复 9 点生命，并清除 1 层流血。

它不能治疗重伤。  
治疗量不能太低，否则修补师治疗评级 B 没有意义；但也不能过高，否则战斗损耗会被轻易抹平。

当前 9 点治疗适合初始血量结构：

- 护路人 68；
- 修补师 46；
- 荒野射手 40。

### 17.2 防护加固说明

防护加固是修补师的护甲流核心技能。

> 防护加固：1 费。目标获得 8 点护甲。护甲在本场战斗内保留，战斗结束后清零。

这张牌可以支持"叠甲流"。  
护甲不是低存在感附赠效果，而是一种可构筑的防御路线。

### 17.3 车体修理说明

车体修理是修补师核心能力。

当前规则：

| 情况 | 效果 |
|---|---|
| 有普通零件 | 消耗 1 普通零件，恢复货车 14 点耐久 |
| 无零件 | 不消耗零件，恢复货车 5 点耐久 |

无零件修车不能太强，但必须能用。  
这样修补师在资源紧张时仍有救急价值。

### 17.4 拆解穿甲说明

拆解穿甲造成 9 点穿甲伤害。

它是修补师少量应对高护甲敌人的工具，不代表修补师有输出能力。  
修补师输出评级仍为 D。

### 17.5 临时补强说明

临时补强是修补师的护盾补救技能。

它和防护加固的区别是：

| 技能 | 防御类型 | 用途 |
|---|---|---|
| 防护加固 | 护甲 | 长期叠甲，适合长战 |
| 临时补强 | 护盾 | 临时挡伤害，适合救急 |

### 17.6 过载说明

过载不是"本回合下一张牌 +50%"。

正确口径：

> 过载：2 费。指定一张牌。该牌下次使用时，数值效果 +100%。使用后，该牌拥有者受到 3 点内伤。

关键点：

| 项目 | 规则 |
|---|---|
| 目标 | 指定一张牌 |
| 持续 | 到该牌下次使用为止 |
| 增幅 | 数值效果 +100% |
| 代价 | 使用后，该牌拥有者受到 3 点内伤 |
| 不是 | 不是本回合下一张牌 |
| 不是 | 不是只加 50% |
| 不是 | 不是一定由修补师受伤 |

例子：

| 情况 | 结算 |
|---|---|
| 修补师对荒野射手的会合射击使用过载 | 会合射击下次使用时效果 +100% |
| 荒野射手之后使用会合射击，原本 22 + 标记额外 6 | 变成 44 + 标记额外 12 |
| 使用后 | 荒野射手受到 3 点内伤 |

再比如：

| 情况 | 结算 |
|---|---|
| 修补师对护路人的稳住阵线使用过载 | 稳住阵线下次使用时护盾效果 +100% |
| 护路人之后使用稳住阵线 | 全体各 8 护盾变成全体各 16 护盾 |
| 使用后 | 护路人受到 3 点内伤 |

## 十九、三种基础战斗路线

第 6.1 的技能表应支持至少三种基础路线。

### 19.1 快攻标记流

核心：荒野射手。

组合：

- 标记；
- 会合射击；
- 补射；
- 过载会合射击。

特点：

- 快速击杀关键敌人；
- 对普通战很有效；
- 射手脆，需要保护；
- 面对高压多目标时容易被打穿。

### 19.2 护车防守流

核心：护路人。

组合：

- 举盾；
- 盾反；
- 稳住阵线；
- 车队防护；
- 临时补强。

特点：

- 保护角色和货车；
- 适合强盗、抢货、攻击货车类敌人；
- 输出较慢；
- 遇到穿甲或状态敌人时不能无脑防守。

### 19.3 叠甲辅助流

核心：修补师。

组合：

- 防护加固；
- 临时补强；
- 车体修理；
- 过载防护加固；
- 护路人稳住阵线。

特点：

- 护甲可在本场战斗内持续叠加；
- 适合精英战、Boss 战等长战斗；
- 战斗结束后护甲清零；
- 害怕穿甲、状态伤害、攻击货车和多目标压力。

## 二十二、6.1 总结

第 6.1 类的核心目标是确定初始三人的战斗结构。

当前结构为：

| 角色 | 核心定位 |
|---|---|
| 护路人 | 主承伤、防护、护车、盾反 |
| 荒野射手 | 脆皮高伤害、标记、流血、侦察 |
| 修补师 | 全能辅助、修车、加固、少量治疗 |

第 6.1 的核心原则是：

> 护路人负责撑住；  
> 荒野射手负责打穿；  
> 修补师负责补救。

三人之间要互相依赖，而不是彼此替代。  
护路人不能成为最高输出，射手不能又脆又无风险，修补师不能变成万能奶妈。

本稿将护甲提升为可以形成流派的战斗资源，将射手重新确立为输出核心，将修补师定位为全能辅助，并将普通战、精英战、Boss 战的节奏目标重新拉开。

后期成长方向不是简单小幅加数值。普通技能可以约翻倍，核心技能和特攻技能可以在条件满足时达到前期 2–3 倍甚至更高。对于后期 Boss 和终局 Boss，可以设计需要多回合准备、多状态叠加、火种或记忆技能支持的极高条件爆发，使玩家在投入资源和承担风险后，能够打出明显改变战局的大伤害。
