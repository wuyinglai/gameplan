# 《余烬商队》第 6.7 类：角色装备、商队模块与道具系统 v0.1

> 文档状态：阶段报告 v0.1  
> 所属大类：第 6 类「角色、职业、技能、记忆与成长系统」  
> 当前用途：确定装备系统的大框架、栏位、商队模块槽、来源、成长方式与旧文明装备规则。  
> 优先级：本报告为当前装备系统最高优先级口径。旧表格、旧聊天记录、旧 README 中与本报告冲突的内容，均降为历史参考。

## 0. 本章核心结论

装备系统不做纯数值装备，也不做每件装备都复杂独特的纯机制装备。

当前原则是：

```text
装备 = 基础数值 + 清楚机制 + 职业限制 + 使用取舍 + 可成长空间
```

本章核心口径：

1. 装备系统分为四大类：角色装备、商队模块、道具系统、特殊装备。
2. 角色装备包括：武器、防具、工具、挂饰。
3. 商队模块是整个商队共用，不绑定单辆车。
4. 商队耐久、资源、订单货损统一结算；不做单车独立耐久，不做单车独立装备树。
5. 基础商队模块栏位为：装甲槽 ×1、车载武器槽 ×1、照明 / 警戒槽 ×2、环境槽 ×1、车体结构槽 ×2、商队凭证槽 ×1、战术道具槽 ×3。
6. 同类型 / 同子类模块互斥，不能同时装备两个完全相同或互斥同类模块。
7. 储存容器不占模块槽，只占商队容量。
8. 灰烬封存箱是储存容器，不是特殊装备。
9. 灰烬是常见但危险的环境 / 材料风险，不应被神话化。
10. 旧文明装备可以残件使用，效果弱；可以逐步修复或启动，恢复更强效果。
11. 普通装备可以多次强化，但本章不写强化曲线；普通装备强化不用火种。
12. 装备不做耐久、不做随机词条、不做传统品质等级。
13. 遗迹探索嵌入跑订单过程，玩家可以边跑订单边探索世界。

## 2. 装备系统四大类

| 大类 | 说明 |
|---|---|
| 角色装备 | 装在角色身上的武器、防具、工具、挂饰 |
| 商队模块 | 装在整个商队上的车队级模块，不绑定单辆车 |
| 道具系统 | 分为战术道具、消耗道具、多次使用工具 |
| 特殊装备 | 旧文明装备、火种启动装备、遗迹修复装备 |

特殊装备不是“更高品质装备”。它的特殊来自来源、修复方式、启动方式和使用代价。

| 物品 | 分类 |
|---|---|
| 防潮箱 | 商队储存容器 |
| 灰烬封存箱 | 商队储存容器，不是特殊装备 |
| 旧文明雾灯 | 特殊装备 / 商队外设 |
| 旧文明车载弩炮 | 特殊装备 / 商队车载武器 |
| 火种启动装置 | 特殊装备 |

## 3. 灰烬、旧文明与装备的关系

### 3.1 灰烬不要神话化

灰烬不是神话资源，也不是一沾上就进入最高级装备系统的东西。

```text
灰烬是常见但危险的环境 / 材料 / 污染风险。
它和受潮、寒湿、污染扩散属于一类生存压力，只是严重程度更高。
```

因此：

| 不要这样写 | 应该这样写 |
|---|---|
| 涉及灰烬就很神秘 | 灰烬是常见风险，只是更危险 |
| 灰烬封存箱算特殊装备 | 灰烬封存箱只是储存容器 |
| 灰烬物品都要火种 | 普通灰烬材料不一定需要火种 |
| 灰烬 = 旧文明 | 灰烬和旧文明可能有关，但不是同一概念 |

### 3.2 旧文明装备才是特殊装备

旧文明装备特点：

| 特点 | 说明 |
|---|---|
| 残件可用 | 残件状态下可使用，但效果弱 |
| 可逐步修复 | 修复后恢复更强效果 |
| 不走普通强化 | 需要旧文明材料、修复方案、专门工匠或设施 |
| 部分需要火种启动 | 强力旧文明装备可消耗火种或特殊能量 |
| 代价明显 | 修复成本高、使用有限制、维护困难 |

例子：旧文明雾灯。

| 阶段 | 效果 |
|---|---|
| 残件状态 | 可用，但只提供弱路线判断 |
| 一阶修复 | 重雾路线中更稳定识别道路 |
| 二阶修复 | 可降低重雾中的伏击和迷路风险 |
| 完全修复 | 效果强，但需要维护材料或特殊滤芯 |

## 6. 角色装备大类

### 6.1 武器类

| 武器大类 | 小类 | 主要标签 | 功能方向 |
|---|---|---|---|
| 盾类 | 小盾、轻盾、重盾 | 护卫 | 防御、护盾、保护商队 |
| 弓类 | 短弓、长弓、猎弓 | 射手 / 猎手 | 标记、远程、稳定输出 |
| 弩类 | 轻弩、重弩、机关弩 | 射手 / 工匠 | 高伤、装填、爆发 |
| 近战类 | 短剑、斧、长柄武器 | 护卫 / 行商 | 拦截、反击、稳定伤害 |
| 工具武器 | 修补锤、撬棍、拆解钳 | 工匠 | 修理、拆解、辅助攻击 |

普通武器初始加成一般不超过 10 点，单件装备初始总数值收益一般不超过 20 点。目的不是让新武器完全碾压旧武器，而是让玩家选择更适合构筑的装备。

#### 例子：重盾

| 项目 | 内容 |
|---|---|
| 分类 | 双手防御武器 |
| 标签 | 护卫 |
| 占用 | 左手 + 右手 |
| 基础数值 | 防御类技能提供的护盾 +3 |
| 机制 | 使用防御类技能时，可额外给商队提供少量护盾或降低本回合商队受击压力 |
| 限制 | 不能装备弓、弩、双手攻击武器；攻击类技能伤害 -2 |

#### 例子：猎弓

| 项目 | 内容 |
|---|---|
| 分类 | 双手攻击武器 |
| 标签 | 射手 / 猎手 |
| 占用 | 左手 + 右手 |
| 基础数值 | 射击伤害 +2 |
| 机制 | 攻击被标记目标时，额外 +2 伤害 |
| 限制 | 对高护甲敌人收益较低 |

#### 例子：重弩

| 项目 | 内容 |
|---|---|
| 分类 | 双手攻击武器 |
| 标签 | 射手 / 工匠 |
| 占用 | 左手 + 右手 |
| 基础数值 | 射击伤害 +5 |
| 机制 | 每场战斗第一次射击类技能额外造成伤害 |
| 限制 | 重弩强化每回合最多触发 1 次，或使用后进入装填限制 |

### 6.2 防具类

防具系统接入已有防御三层。

| 防御层 | 定位 | 装备接口 |
|---|---|---|
| 护盾 | 临时防护，战斗内常见 | 轻甲、盾牌、战术道具可提供 |
| 护甲 | 更稳定的长期防护层 | 防具和车辆装甲最适合提供 |
| 屏障 | 高级防护，稀有强力 | 旧文明装备、特殊装备少量提供 |

防具可以提供：开局护盾、开局护甲、减伤、受潮抵抗、寒湿抵抗、灰蚀抵抗、特定地形 / 天气惩罚降低。

防具不要写：降低重伤风险、防止重伤、生命归零不重伤、无限续命。生命归零进入重伤是已有规则，装备不能随意破坏。

| 防具大类 | 小类 | 作用 |
|---|---|---|
| 头部 | 头盔、护目镜、面罩 | 护甲、雾中观察、灰烬防护 |
| 躯干 | 轻甲、中甲、重甲、防护服、防潮披风 | 护甲、护盾、天气抵抗 |
| 手部 | 手套、护手、工具手套 | 工具效率、受伤保护 |
| 腿部 | 护腿、绑腿 | 行路事件、迟缓抵抗 |
| 足部 | 靴子、泥地靴、防寒靴 | 地形、天气、行路风险 |

### 6.3 工具类

工具不要用“挎包”“腰包”这类像单纯加容量的名字，改成具体器具。

| 工具 | 标签 | 功能方向 |
|---|---|---|
| 包扎器具组 | 医师 | 提高治疗技能或药包效果 |
| 夹板固定组 | 医师 | 战斗外处理伤势，提高休息回血 |
| 维修工具箱 | 工匠 | 提高零件、修理、临时修复效果 |
| 拆解工具组 | 工匠 | 对护甲、机械、旧文明装置有额外处理 |
| 侦察镜 | 猎手 / 行商 | 获得免费粗略传闻 |
| 测灰器 | 灰烬 / 工匠 | 判断灰烬污染、灰烬材料风险 |
| 交易凭牌 | 行商 | 交易、聚落交涉、价格稳定 |

### 6.4 挂饰类

挂饰不要太强，主要做轻量被动和事件接口。

| 挂饰 | 方向 |
|---|---|
| 护符 | 士气、灰烬污染、特殊事件 |
| 小型凭证 | 聚落交涉、驿站服务 |
| 旧世铭牌 | 遗迹识别、旧文明设备互动 |
| 信物 | 特定城市、订单、势力相关加成 |

## 7. 商队模块系统

### 7.1 核心结算口径

多辆车不做独立装备系统。车辆装备统一改为商队级模块系统。

```text
所有车辆共同组成商队。
商队耐久统一结算。
资源统一结算。
订单货损统一结算。
车辆装备作用于整个商队。
不做每辆车独立耐久。
不做每辆车独立装备树。
```

如果每辆车独立装备，会产生明显问题：A 车装护甲为什么保护 B 车、B 车装宽轮为什么 A 车也不坏、C 车装防潮箱到底保护哪辆车的货。这些都会迫使系统做单车结算，复杂度太高。

因此本章采用商队整体装备系统。多辆车只影响商队容量、商队耐久上限、模块槽数量、重量和维护成本。

## 8. 商队模块栏位

### 8.1 基础商队模块栏位总表

| 槽位 | 数量 | 可装内容 | 规则 |
|---|---:|---|---|
| 装甲槽 | 1 | 装甲板、防撞板、护货挡板 | 只能选 1 个 |
| 车载武器槽 | 1 | 车载弩炮、车载弩、车载投射器 | 只能选 1 个，通常额外占容量 |
| 照明 / 警戒槽 | 2 | 雾灯、警戒铃、照明器、警戒旗、探照灯 | 同类型互斥，不能装两个完全相同或同子类互斥装备 |
| 环境槽 | 1 | 防潮板、遮阳棚、防寒罩、灰烬滤网 | 只能选 1 个 |
| 车体结构槽 | 2 | 车轮、车架、减震、牵引、制动类改装 | 可装 2 个不同子类，同类型互斥 |
| 商队凭证槽 | 1 | 商牌、路引、通行凭证、势力证明 | 只能选 1 个 |
| 战术道具槽 | 3 | 绊索器、信号枪、便携盾架、探路杆组 | 不一定占容量，但必须占槽位 |

### 8.2 同类型互斥规则

商队模块采用“同类型互斥”规则。

```text
两个同类型 / 同子类的模块不能同时装备。
不能装备两个完全相同模块。
不能通过重复装备同一类模块叠加同类效果。
```

例子：

| 搭配 | 是否允许 | 原因 |
|---|---|---|
| 宽轮 + 加固车架 | 允许 | 车轮类 + 车架类，不同子类 |
| 宽轮 + 减震结构 | 允许 | 车轮类 + 减震类，不同子类 |
| 加固车架 + 强化牵引架 | 允许 | 车架类 + 牵引类，不同子类 |
| 宽轮 + 泥地轮 | 不允许 | 都是车轮类，互斥 |
| 减震结构 + 软轴结构 | 不允许 | 都是减震类，互斥 |
| 加固车架 + 加固车架 | 不允许 | 完全相同 |
| 雾灯 + 警戒铃 | 允许 | 照明 / 警戒槽中不同子类 |
| 雾灯 + 旧文明雾灯 | 不允许或需特殊替换规则 | 同属雾灯类，互斥 |
| 警戒铃 + 警戒铃 | 不允许 | 完全相同 |

### 8.3 各槽位说明

#### 装甲槽 ×1

装甲槽用于商队防御。

| 装备 | 效果方向 |
|---|---|
| 装甲板 | 战斗开始给商队护甲 |
| 防撞板 | 路况事件中减少商队耐久损失 |
| 护货挡板 | 降低货损 |

装甲槽只能选一个，玩家要在抗战斗、抗路况、保货物之间取舍。

#### 车载武器槽 ×1

车载武器槽用于车队战斗火力。

| 装备 | 效果方向 |
|---|---|
| 车载弩炮 | 每场战斗提供一次车载射击 |
| 轻型车载弩 | 伤害较低，但占用小、弹药便宜 |
| 重型车载弩炮 | 伤害高，占容量大，弹药贵 |
| 车载投射器 | 发射燃烧瓶、爆裂弹等特殊弹药 |

车载武器通常额外占商队容量，部分还需要弹药。装车载武器等于少装货，换战斗火力。

#### 照明 / 警戒槽 ×2

照明 / 警戒槽用于侦察、夜间、重雾、遗迹和伏击预警。

| 装备 | 作用 |
|---|---|
| 雾灯 | 重雾中改善路线判断，降低伏击 / 迷路压力 |
| 照明器 | 夜间、遗迹、灰暗区域探索辅助 |
| 警戒铃 | 降低夜间偷袭、偷货、伏击压力 |
| 警戒旗 | 更容易被友方 / 商队识别，降低误判冲突 |
| 探照灯 | 遗迹探索、夜间事件中提供更强信息 |

照明 / 警戒槽有 2 个，但同类型互斥。

#### 环境槽 ×1

环境槽用于应对天气、灰烬、寒冷、炎热和潮湿。

| 装备 | 效果方向 |
|---|---|
| 防潮板 | 降低雨天受潮风险 |
| 遮阳棚 | 酷热天气降低补给压力 |
| 防寒罩 | 严寒天气降低恢复惩罚 |
| 灰烬滤网 | 灰烬风 / 灰烬污染区域降低污染扩散 |

环境槽只有 1 个，玩家要根据路线情报选择环境适配方向。

#### 车体结构槽 ×2

车体结构槽用于商队行路、稳定性、车架和道路适配。

| 子类 | 例子 | 作用 |
|---|---|---|
| 车轮类 | 宽轮、泥地轮、铁箍轮 | 应对泥地、湿地、雨后道路 |
| 车架类 | 加固车架、轻量车架 | 改变商队耐久上限或重量 |
| 减震类 | 减震结构、软轴结构 | 应对山地、烂路、遗迹道路 |
| 牵引类 | 强化牵引架、双挂牵引 | 降低绕路、坡地、拖拽压力 |
| 制动类 | 加强制动、坡道制动器 | 降低下坡、湿滑道路风险 |

车体结构槽有 2 个，可以装两个不同子类；同类型 / 同子类互斥。

#### 商队凭证槽 ×1

商队凭证槽吸收原“车队牌”的部分功能。

| 凭证 | 效果方向 |
|---|---|
| 商牌 | 某些城市 / 城镇交易更稳定 |
| 路引 | 特定路线通行、事件交涉 |
| 驿站通行凭证 | 驿站服务或情报更方便 |
| 势力证明 | 特定订单、势力事件有额外选项 |

基础商队只能挂 1 个凭证，玩家要选择这趟路线使用哪种身份或凭证。

#### 战术道具槽 ×3

战术道具可以不占商队容量，但必须占槽，不能无限携带。

| 战术道具 | 使用限制 |
|---|---|
| 绊索器 | 每场战斗 1 次 |
| 信号枪 | 每条路线段 1 次，需要信号弹 |
| 便携盾架 | 每场战斗 1 次 |
| 探路杆组 | 路况事件中每路线段 1 次 |
| 警戒哨 | 驻留 / 夜间事件中触发 |

## 9. 储存容器

储存容器不占模块槽，只占商队容量。

| 储存容器 | 占用 |
|---|---|
| 防潮箱 | 商队容量 |
| 灰烬封存箱 | 商队容量 |
| 药品箱 | 商队容量 |
| 材料箱 | 商队容量 |
| 特殊货物箱 | 商队容量 |

例子：

```text
防潮箱：占 2 点商队容量，可保护一定数量怕潮货物。
灰烬封存箱：占 2 点商队容量，可封存灰烬材料或污染货物。
```

储存容器不是模块槽装备，不和装甲槽、环境槽等冲突。它的代价是占用商队容量，减少可装订单货物或资源。

## 10. 商队模块例子

| 模块 | 分类 | 占用 | 主要效果 | 限制 |
|---|---|---|---|---|
| 装甲板 | 装甲装备 | 装甲槽 | 战斗开始时商队获得 10 点商队护甲 | 增加重量，占唯一装甲槽 |
| 宽轮 | 车体结构 / 车轮类 | 车体结构槽 | 雨后泥地、湿地、烂路事件中降低商队耐久损耗 | 与其他车轮类互斥 |
| 防潮板 | 环境装备 | 环境槽 | 雨天货物受潮风险降低 | 只对雨天 / 潮湿事件有效 |
| 防潮箱 | 储存容器 | 商队容量 | 箱内怕潮货物不受雨天受潮影响 | 只保护箱内容量 |
| 灰烬封存箱 | 储存容器 | 商队容量 | 封存灰烬材料或污染货物，降低污染扩散风险 | 不保护普通货物，不算特殊装备 |
| 车载弩炮 | 车载武器 | 车载武器槽 + 容量 | 每场战斗发动 1 次车载射击 | 占容量，伏击首回合不可用，可需要弹药 |

## 11. 道具系统

道具系统分三类。

| 类型 | 定义 | 例子 |
|---|---|---|
| 战术道具 | 不直接消失，但有次数、冷却、装填、弹药或路线段限制 | 绊索器、信号枪、便携盾架 |
| 消耗道具 | 用一次少一个，或使用后消失 | 药包、燃烧瓶、爆裂箭、烟雾罐、净灰瓶 |
| 多次使用工具 | 可重复使用，通常用于事件或路线 | 遮雨布、绳索、探路杆 |

修正口径：

```text
烟雾罐明显是一次性消耗品，不归战术道具。
遮雨布明显可重复使用，不归一次性消耗品。
不做“防止重伤”道具。
生命归零仍按重伤规则处理。
```

### 11.1 战术道具大类

| 类型 | 作用 |
|---|---|
| 控制类 | 迟缓、限制敌人、打断行动 |
| 支援类 | 帮助撤退、增加治疗量、提供临时护盾 |
| 侦察类 | 降低伏击风险、获得免费传闻 |
| 车队类 | 保护商队耐久、保护货物 |
| 环境类 | 应对天气、地形、灰烬污染 |

### 11.2 战术道具例子

| 道具 | 类型 | 限制 | 效果 |
|---|---|---|---|
| 绊索器 | 战术道具 | 每场战斗 1 次 | 对一个地面敌人施加迟缓 |
| 信号枪 | 战术道具 | 每条路线段 1 次，需要信号弹 | 降低下一次伏击风险，或在撤退事件中增加撤退进度 |
| 便携盾架 | 战术道具 | 每场战斗 1 次 | 给商队或一名角色提供临时护盾，不能防止重伤 |

### 11.3 消耗道具例子

| 道具 | 类型 | 效果 | 限制 |
|---|---|---|---|
| 烟雾罐 | 消耗道具 | 本回合降低敌方远程伤害，伏击战中可增加撤退窗口 | 使用后消耗 1 个，雨天 / 灰烬风中效果降低 |
| 燃烧瓶 | 消耗道具 | 造成火焰伤害，并施加灼烧 | 雨天火焰伤害按天气规则降低 |
| 爆裂箭 | 消耗道具 | 射手技能可消耗爆裂箭造成额外伤害或小范围伤害 | 需要弓 / 弩类装备，无射手标签不能完整使用 |

## 12. 特殊装备：旧文明装备与火种启动装备

### 12.1 特殊装备定位

特殊装备主要指：

| 类型 | 说明 |
|---|---|
| 旧文明装备 | 遗迹、废墟、生产基地中获得的残件或设备 |
| 火种启动装备 | 需要火种或特殊能量启动的强力装备 |
| 遗迹修复装备 | 需要修复方案、旧文明材料、专门工匠修复 |

特殊装备可以是角色装备、商队模块，也可以是道具。

### 12.2 旧文明装备规则

| 规则 | 内容 |
|---|---|
| 残件可用 | 残件状态下可使用，但效果弱 |
| 可逐步修复 | 通过材料、图纸、专门工匠恢复效果 |
| 不走普通强化 | 不用普通金币材料强化 |
| 修复代价高 | 需要旧文明残件、修复方案、特殊材料 |
| 收益明显 | 完成修复后效果要足够强 |
| 部分可火种启动 | 强力装备可消耗火种获得特殊效果 |

### 12.3 例子：旧文明车载弩炮

| 阶段 | 效果 |
|---|---|
| 残件 | 每场战斗可进行一次不稳定车载射击 |
| 一阶修复 | 车载射击稳定，伤害提高 |
| 二阶修复 | 可和标记 / 穿甲类效果互动 |
| 火种启动 | 关键战斗中发动更强射击，但消耗火种能量或特殊弹药 |

## 13. 装备成长与强化

### 13.1 普通装备成长

普通装备可以多次强化，但本章不写具体曲线。

| 强化方向 | 消耗 | 作用 |
|---|---|---|
| 数值强化 | 金币 + 普通材料 | 提高伤害、护盾、治疗、修理等基础数值 |
| 结构强化 | 金币 + 稀有材料 | 提高机制次数、范围或稳定性 |
| 适配强化 | 职业材料 / 训练设施 | 让装备更适合某职业 |
| 图纸改造 | 图纸 + 材料 + 金币 | 解锁新功能或分支 |

普通装备不用火种强化。火种主要用于技能词条、记忆相关系统，以及部分旧文明装备启动。

不做：随机词条、传统品质等级、装备耐久、普通装备火种强化、无限数值膨胀。

### 13.2 旧文明装备成长

旧文明装备不叫强化，叫修复、启动、适配。

| 方式 | 说明 |
|---|---|
| 修复 | 恢复旧文明装备原有效果 |
| 启动 | 让沉睡或残缺设备重新工作 |
| 适配 | 让角色或商队能够安全使用 |

旧文明装备需要残件、修复图纸 / 方案、旧文明材料、专门工匠 / 城市设施。火种只用于部分强力装备启动。

## 14. 装备来源

| 来源 | 产出 |
|---|---|
| 城市商店 | 普通角色装备、基础商队模块、普通图纸 |
| 城镇商店 | 少量人物基础装备，如铁剑、普通护甲 |
| 驿站商店 | 少量商队模块，如装甲板、外设、环境装备 |
| 城市工坊 | 制作、强化、修复 |
| 订单奖励 | 装备、材料、凭证、特殊图纸 |
| 战斗 | 金币、材料、小概率装备 |
| 精英 / Boss | 高价值装备、图纸、旧文明残件 |
| 特殊事件 | 破损装备、图纸、旧文明线索 |
| 遗迹探索 | 旧文明装备、修复材料、特殊图纸 |

城镇和驿站不做完整装备中心：城市是主要装备购买、制作、强化点；城镇少量普通人物装备；驿站少量普通商队模块；遗迹产出旧文明装备、稀有材料、特殊图纸。

### 14.1 大致掉落 / 产出概率

| 来源 | 装备 / 图纸 / 模块产出概率 |
|---|---:|
| 普通战斗 | 10%–20% 出基础装备、消耗道具或小材料 |
| 精英战斗 | 40%–70% 出装备、图纸、模块或高价值材料 |
| Boss / 关键战斗 | 100% 出高价值奖励，包含装备、火种、模块、图纸之一 |
| 普通事件 | 10%–30% 出道具、材料、破损装备 |
| 高风险事件 | 30%–60% 出装备、图纸、特殊道具 |
| 特殊订单 | 50%–100% 按订单类型给装备、图纸或凭证 |
| 遗迹探索 | 中高概率获得材料、图纸、旧文明残件 |

普通战斗不能变成刷装备，但战斗收益也不能太低，否则玩家会无脑避战。精英、Boss、遗迹才是装备和图纸的重要来源。

## 15. 遗迹探索与订单路线

遗迹探索不是完全独立玩法，而是嵌入跑订单过程。

玩家跑订单时，会在路线中遇到遗迹、废墟、旧文明生产基地、被毁城市、地下仓库、旧道路枢纽等探索点。

如果订单时间充裕、补给够、商队状态允许，玩家可以选择进入遗迹探索。探索会消耗天数、补给、药包、零件、商队耐久和订单时间压力，但可能获得金币、材料、图纸、旧文明残件、特殊装备、情报和路线线索。

探索结束后，玩家继续原订单路线。

```text
玩家可以边跑订单，边探索世界。
```

| 遗迹类型 | 可能收益 | 主要风险 |
|---|---|---|
| 被毁城市 | 旧文明残件、金币、图纸、材料 | 敌人多、路线复杂 |
| 废弃生产基地 | 商队模块、制造图纸、零件材料 | 机械敌人、商队耐久损耗 |
| 灰烬污染遗迹 | 灰烬材料、封存货物、旧文明线索 | 灰蚀、污染扩散 |
| 旧道路枢纽 | 路线情报、旧世路引、车载设备 | 伏击、迷路 |
| 地下仓库 | 储存容器、特殊货物、装备残件 | 补给消耗、特殊敌人 |

灰烬污染遗迹只是遗迹类型之一，不是神话终极区。遗迹探索是风险收益选择，不是主线必须完成的副本。

## 16. 装备制作

普通装备不一定需要图纸。

| 装备类型 | 是否需要图纸 |
|---|---|
| 普通剑、普通弓、普通护甲 | 不需要特殊图纸，城市工坊默认会 |
| 常见商队模块 | 普通版不需要特殊图纸 |
| 复杂装备 | 需要图纸 |
| 职业专属装备 | 需要图纸或记忆解锁 |
| 旧文明装备 | 必须需要修复图纸 / 修复方案 |
| 火种启动装备 | 必须需要图纸 / 特殊设施 |

制作要素：金币、普通材料、稀有材料、图纸、旧文明残件、专门工匠 / 城市设施。

例子：

| 装备 | 需求 |
|---|---|
| 普通弓 | 木材 + 兽筋 + 金币，不需要特殊图纸 |
| 防潮箱 | 防潮布 + 木材 + 金币，普通版不需要特殊图纸 |
| 高级防潮箱 | 防潮箱图纸 + 更好材料 + 金币 |
| 旧文明雾灯 | 旧世透镜 + 雾灯残件 + 修复图纸 + 城市工坊 |

## 17. 商队牌 / 车队牌处理

车队牌暂时不单独做大系统。

| 原车队牌内容 | 处理方式 |
|---|---|
| 通行牌、商牌、路引 | 商队凭证槽 / 挂饰 / 特殊道具 |
| 主动战术牌 | 尽量少做，控制在 5 张以内 |
| 战斗牌效果 | 转成战术道具或消耗道具 |
| 车辆牌效果 | 转成商队模块特殊效果 |
| 势力证明 | 转成凭证类挂饰或订单道具 |

理由：车队牌如果做太大，会和装备、道具、订单系统重叠。当前只保留轻量接口，不作为第 6.7 的核心大系统。

## 18. 本章总口径

```text
1. 装备系统属于第 6 类，暂命名为第 6.7 类。

2. 装备系统分为：角色装备、商队模块、道具系统、特殊装备。

3. 角色装备栏包括：左手、右手、头部、躯干、手部、腿部、足部、工具栏、挂饰栏。

4. 武器按左右手占用：单手装备占 1 手，双手装备占 2 手；重盾是双手防御武器。

5. 装备有职业标签：匹配标签才能完整使用，不匹配不能用或只能用基础效果。

6. 防具接入已有防御三层：护盾、护甲、屏障；不写降低重伤风险，不写防止重伤。

7. 商队模块不绑定单辆车。所有车辆共同组成商队，统一商队耐久、资源、货损结算。

8. 基础商队模块栏包括：装甲槽×1、车载武器槽×1、照明/警戒槽×2、环境槽×1、车体结构槽×2、商队凭证槽×1、战术道具槽×3。

9. 同类型 / 同子类模块互斥，不能装备两个完全相同或互斥的同子类模块。

10. 储存容器不占模块槽，占商队容量。防潮箱、灰烬封存箱都是储存容器。

11. 灰烬封存箱不是特殊装备。灰烬是常见但危险的环境 / 材料风险，不是神话资源。

12. 特殊装备主要指旧文明装备、火种启动装备、遗迹修复装备。

13. 旧文明装备可以残件使用，效果弱；可以逐步修复，恢复原有效果；不走普通强化。

14. 普通装备可以多次强化，但本章不写强化曲线。普通装备强化不用火种。

15. 装备不做装备耐久、随机词条、传统品质等级、单车独立装备树。

16. 道具分为战术道具、消耗道具、多次使用工具。

17. 遗迹探索嵌入跑订单过程，玩家可以边跑订单边探索世界。
```
