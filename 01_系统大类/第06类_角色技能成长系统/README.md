## 6.2 权威口径摘要

- 轻伤不做，只保留重伤。
- 队员生命值降到 0 或以下后，本场战斗重伤退出。
- 重伤退出后，不能再使用该队员专属牌，也不再抽取该队员的 2 张专属牌。
- 每回合抽牌数 = 当前可行动队员数 × 2 + 1 张车队牌。
- 战斗终止条件不是全员重伤本身，而是"所有出战队员均重伤退出 + 货车耐久归零"同时满足。
- 货车耐久 0 是抛锚，不是立刻失败；必须修到 20 以上才能继续移动。
- 一次重伤：3 天恢复，最大生命 -20%，恢复到当前最大生命 20%，向上取整。
- 二次重伤：4 天恢复，最大生命累计 -50%，恢复到当前最大生命 20%，向上取整。
- 第三次重伤进入失能，不能上场，但可以城市治疗。
- 驿站 / 城镇可以加速重伤自然恢复，但不能清除重伤次数，也不能恢复最大生命惩罚。
- 城市可以治疗一次重伤、二次重伤、失能；治疗统一花费 1 天和金币。
- 正常游玩不做永久伤残、永久死亡或不可逆惩罚。
- 伤员恶化只让当天不恢复并使剩余恢复时间 +1 天，不升级重伤次数。
- 内伤是不可阻挡伤害，不是长期疾病；外伤是可被防御处理的伤害类型。
- 灰蚀 3 层爆发造成 8 点内伤。
- 中毒是长期异常，战斗结束后层数减半，向上取整保留。
- 不做经验值、熟练度、技能碎片、记忆碎片、创伤记忆、重伤觉醒。
- 记忆是旧文明技术，完整获得，要么获得完整记忆，要么不获得。
- 记忆分为攻击、防御、辅助、消耗四类。
- 记忆和普通技能共享出战技能 / 记忆位；初始 6 个，中期可成长到 7，后期可成长到 8。
- 记忆可以金币小幅强化、火种特殊强化，也可以用完整记忆固定融合。
- 招募类型只有同职业变体、新职业、新职业变体；不做临时雇员。
- 商队规模不是升级等级，而是本趟出行配置和长期运营能力分类。
- 小型出行：3 人，3 行动力，抽 7，每日补给 1。
- 中型出行：4 人，4 行动力，抽 9，每日补给 2。
- 大型出行：5 人，5 行动力，抽 11，每日补给 3。
- 不硬性禁止玩家带更多人，而是通过补给消耗、车厢格子、货物空间和路线距离形成软限制。
- 普通补给 1 格、重量 1、不堆叠；压缩干粮 1 格、重量 1、支撑 2 天；军用口粮 1 格、重量 1、支撑 3 天。
- 初始车队 2 辆基础货车，每辆 8×8 车厢格，共 64 格，初始总车厢 128 格。
- 普通零件占 1 格、重量 2、不堆叠。
- 车厢容量成长目标：初始 128 → 第一座城市升级后 200 → 中期 320 → 后期 450 → 最终 600 格。
- 初始车厢只支持短程商路（约 60 天），不要求支持中程（约 100 天）、长程（约 150 天）或超长程（约 200 天）。
- 旧口径"每辆 6×7，总计 84 格"已废弃；旧口径"初始 200 格"已废弃，200 格改为第一座城市升级后目标。

## 6.3 权威口径摘要

v0.2 替换 v0.1 旧口径。新版 6.3 采用每名角色 4 个能力槽：2 个数值被动 + 2 个事件 / 世界接口。数值被动固定生效，不额外叠触发概率；事件接口只在对应事件中出现，不是常驻主动技能。

- 每名角色拥有 4 个能力槽：2 个数值被动 + 2 个事件 / 世界接口。
- 数值被动固定生效，影响权重、信息、修车、资源、损失等，不再额外叠触发概率。
- 事件接口不是常驻主动技能，只在满足条件的事件中出现额外选项、提示或结果修正。
- 初始三人能力槽：
  - 护路人：护卫 / 护车（数值被动）+ 挡在前面 / 维持秩序（事件接口）；
  - 荒野射手：野外侦察 / 狩猎采集（数值被动）+ 看懂痕迹 / 野外取食（事件接口）；
  - 修补师：修理知识 / 现场诊断（数值被动）+ 临场修补 / 伤病判断（事件接口）。
- 信息系统采用 0–5 级，初始角色免费信息最高到 2 级。
- 3–5 级信息留给装备、工具、路线情报、记忆、后续专精角色。
- 6.3 不做战斗技能成长、不做能力槽升级、不做完整事件文本。
- 旧版 v0.1"每人只有 2 个基础词条"已废弃，不再作为当前口径。

## 6.4 权威口径摘要

- 初始角色技能成长分为普通升级和火种升级两条独立线。
- 普通升级由金币和设施支持，分为数值升级（伤害+1、护盾+1、治疗+1、修车+1 等）与状态升级（标记层数+1、流血层数+1、净化层数+1 等）。
- 火种升级独立于普通等级，用于添加词条 / 后缀 / 特殊效果；每个技能有自己的火种词条池，不随机。
- 技能融合不设主技能 / 副技能；融合后分别继承参与技能中最高的数值升级点和最高的状态升级点，并允许玩家重新分配。
- 火种词条融合时原则上尽量保留，但必须通过新技能词条池检查。
- 游戏里没有"记忆碎片"，只叫"记忆"；记忆可通过融合转化为主动技或被动技。
- 不写"强化标记"等笼统表达，改写成具体升级项。
- 当前不设计削弱护甲机制；若需要相关效果，只写"穿甲 / 无视护甲"。
- 升级价格本版不定，后续在全局物价框架确定后再定。
- 技能成长主要依赖城镇 / 城市设施，不做野外随地升级。

## 6.7 权威口径摘要

第6.7类_角色装备商队模块与道具系统_v0.1.md 是当前装备系统最高优先级口径。旧表格、旧聊天记录、旧 README 中与本报告冲突的装备、车辆模块、车队牌、道具、旧文明装备口径，均降为历史参考。

- 装备系统分为四大类：角色装备、商队模块、道具系统、特殊装备。
- 角色装备栏包括：左手、右手、头部、躯干、手部、腿部、足部、工具栏、挂饰栏。
- 武器按左右手占用：单手装备占 1 手，双手装备占 2 手，重盾是双手防御武器。
- 装备有职业标签，匹配标签才能完整使用。
- 防具接入已有防御三层：护盾、护甲、屏障。不写降低重伤风险、防止重伤。
- 商队模块不绑定单辆车；商队耐久、资源、订单货损统一结算；不做单车独立耐久和单车独立装备树。
- 基础商队模块栏位：装甲槽×1、车载武器槽×1、照明/警戒槽×2、环境槽×1、车体结构槽×2、商队凭证槽×1、战术道具槽×3。
- 同类型/同子类模块互斥，不能装备两个完全相同或互斥同子类模块。
- 储存容器不占模块槽，只占商队容量。防潮箱、灰烬封存箱、药品箱、材料箱都是储存容器。
- 灰烬封存箱不是特殊装备。灰烬是常见但危险的环境/材料风险，不要神话化。
- 特殊装备主要指旧文明装备、火种启动装备、遗迹修复装备。旧文明装备可残件使用，逐步修复。
- 普通装备可多次强化，但 v0.1 不写强化曲线。普通装备强化不用火种。
- 装备不做装备耐久、随机词条、传统品质等级、单车独立装备树。
- 道具分为战术道具、消耗道具、多次使用工具。烟雾罐是消耗道具，遮雨布是多次使用工具。
- 车队牌不单独做大系统，通行牌/商牌转商队凭证槽，主动战术牌控制在 5 张以内。
- 遗迹探索嵌入跑订单过程，玩家可以边跑订单边探索世界。
