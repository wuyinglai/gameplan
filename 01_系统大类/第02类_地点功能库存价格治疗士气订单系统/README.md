## 2.1 权威口径摘要

- 第2.1类_全局经济与物价锚点系统_v0.1.md：当前全局经济锚点，覆盖金币面额、基础物资价格、城市/城镇/驿站经济定位、治疗修理、路线消耗、订单收益、情报价格、技能升级、记忆融合与火种词条经济成本。
- 城市是唯一真正的大安全点；城镇和驿站只是路上的补给容错点。
- 统一叫"商队耐久"；不写车况、车框、货车耐久；修车 = 恢复商队耐久。
- 基础物价：城市补给10/城镇15/驿站16，城市零件50/城镇75/驿站70，城市药包50/城镇70/驿站75。
- 普通用零件恢复10商队耐久；有修补师恢复15（不是16）。
- 安全点修理：城市10金/20耐久，驿站25金/20耐久，城镇30金/20耐久。
- 普通药包恢复10生命；有修补师恢复15生命。
- 安全点治疗：城市10金/20生命，城镇25金/20生命，驿站30金/20生命。
- 重伤/失能只有城市可恢复：第一次重伤200，第二次重伤350，失能500，整队失能总成本不超过500。
- 路线消耗公式：天数 × 30 金币。
- 单个订单收益 ≈ 预计消耗 × 1.4–1.7；长线真实收益约20000–30000。
- 情报分精度+可信度两层；免费传闻50%–60%可信，付费情报最低70%可信。
- 数值升级300起每次+20；状态升级1000起每次+200。
- 同标签记忆融合每个300金币；跨标签每个500金币。
- 火种词条：第1个1火种+300金币，递增。
- 游戏里没有"记忆碎片"，只叫"记忆"。
