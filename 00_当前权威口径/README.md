| 目录 | 用途 | 优先级 |
|---|---|---|
| 01_系统大类/ | 当前系统阶段报告 | 最高 |
| 00_当前权威口径/ | 目录、优先级、修订规则 | 最高 |

## 口径冲突判断

1. 同一系统内，修改日期越新优先级越高。
2. 明确标注"当前权威口径 / 阶段报告 / 封档"的文件优先。
3. `01_系统大类/` 下的新阶段报告优先于根目录旧 txt / xlsx / docx。
4. 新报告已经明确废弃的旧口径，不再作为有效规则。
5. 无法判断的冲突，写入 `旧口径清理清单.md`，等待确认。
