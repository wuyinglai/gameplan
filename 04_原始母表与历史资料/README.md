# 04 原始母表与历史资料

本目录用于存放《余烬商队》的原始 Excel 母表、早期 txt 资料和历史版本文件。

## 说明

本目录为历史资料归档区。当前不移动用户原始文件，仅建立目录和索引。后续整理时可逐步将根目录的 xlsx、txt、docx 文件迁入。

## 优先级

最低。如与 `01_系统大类/` 下的阶段报告冲突，以阶段报告为准。
